# 风格规范：一级市场行业研究报告

## 参考文件

- PDF1：TMT 深度研究报告参考（21 页，风格 A）
- PDF2：消费行业研究报告参考（49 页，风格 B）

## 页面构成与布局

### 风格 A（深度讨论型 / TMT 聚焦）
- **页边距**：宽松（约 2.5cm 四周）
- **页眉**：机构 logo（右上角），图标 + 机构名称（双语），小号字体
- **页脚**：无
- **信息密度**：中等，以观点性文字为主
- **首页**：完整标题区（标题、副标题、作者、日期、二维码），紧接带边框段落的执行摘要
- **行距**：正文约 1.5 倍

### 风格 B（传统行业型 / 消费/医疗健康）
- **页边距**：标准（约 2cm 四周）
- **页眉**：报告标题居中置顶，下方为细黑色横线——所有内容页均需包含，包括第一个内容页
- **页脚**：页码（居中或右对齐）
- **信息密度**：较高，图表频繁出现
- **首页**：封面仅含居中标题、标题下方居中 logo、机构名称 + 日期居中置于页面底部
- **第二页**：目录，三级点线引导层级，含页码
- **行距**：正文约 1.5 倍
- **页眉适用范围**：目录之后的所有内容页，包括含图表的页面

## 图表视觉处理

### 表格——风格 A
- 浅色边框，简洁外观
- 基准对比表用于数据驱动分析
- 标题在上方："表 N XXX"
- 来源在下方："数据来源：xxx"

### 表格——风格 B（极其重要）
- **列间无竖线**
- **顶部边框**：粗黑线贯穿整个表格
- **表头行**：上下以细水平线分隔
- **数据行**：行间以细水平线分隔
- **底部边框**：粗黑线贯穿整个表格
- **表头和数据单元格无背景色**（全白）
- 标题在表格上方："表 N XXX"
- 来源注释在表格下方："数据来源：xxx"或"资料来源：xxx"
- 字体：与正文相同（楷体/华文楷体），单元格内容字号略小
- 文字对齐：文本列左对齐，数字列居中

### 图表——风格 A
- **原生图表**为主要可视化方式，非截图
- **柱状图**：分组或堆叠柱状图，用于市场规模、收入拆分、竞品对比；坐标轴简洁，不使用 3D 效果
- **折线图**：用于增长趋势、渗透率曲线、技术性能随时间的演变
- **饼图/环形图**：用于市场份额、收入构成、用户分层
- **雷达图/蛛网图**：用于多维度技术或产品能力对比
- **瀑布图**：用于收入桥接分析、成本结构拆解
- **时间线/路线图**：用于技术演进、产品发布节奏、监管里程碑
- **架构/流程图**：用于技术栈、价值链、平台生态映射
- **基准对比表**：主要用于 AI/模型研究——模型性能对比（准确率、延迟、成本、参数量）；样式为带粗体表头的数据表格
- 标题格式："图 N [描述]"，置于图表下方
- 来源注释："数据来源：xxx"，置于标题下方
- 配色：蓝色（#4A90D9）、青色（#2E8B8B）、橙色（#E8913A）、绿色（#5BA55B）、灰色（#808080）——柔和的机构风格色调，非鲜艳亮色
- 图表密度：每 2-3 页 1-2 个图表（多于纯文本，少于风格 B）

### 图表——风格 B
- 饼图、柱状图、折线图，使用简洁的机构风格配色
- **标题格式**："图 N XXX"，置于图表上方
- **来源注释**：图表下方："数据来源：xxx"
- 图表风格：简洁，极少网格线，扇形区域标注百分比
- 图表配色：青/绿/灰/蓝色调（非鲜艳亮色）
- 相关图表可并排显示（同一行两个图表），共用标题格式
- 产业链图：彩色矩形框搭配连接箭头

### 外部视觉内容（仅风格 A）
- 引用外部产品 UI 或界面时，使用描述性示意图或标注插图替代截图
- 视觉内容下方添加标题
- 需要强调处可使用红框高亮

## 封面设计（关键要求）

### 风格 A 封面
1. **白色背景**——无色块面板、无渐变
2. **机构 logo 区域**（右上角）：
   - 图标 + 机构名称（主语言）+ 机构名称（副语言）置于下方
3. **标题区**（左至居中）：
   - 主标题第一行："[报告标题]"，楷体加粗，黑色
   - 主标题第二行："[副标题]"，楷体加粗，黑色
   - 标题下方细水平装饰线（深红色）
4. **作者行**："Cr.: [作者姓名]"，置于标题区下方
5. **日期行**："Date: YYYY-Mon"，置于作者行下方
6. **二维码**（右侧区域，logo 下方）

### 风格 B 封面
1. **白色背景**——极简风格，无装饰元素
2. **完整标题**：水平居中，楷体，大字号（20-24pt）
   - 标题较长时可分两行
3. **机构 logo**：居中置于标题下方，中小尺寸
4. **机构名称**：如"[机构名称]研究院"——居中，较小字号，置于 logo 下方
5. **日期**：如"2023.12"——居中，置于页面底部区域
6. **无装饰线条、边框或背景色**

## 页眉/页脚规则（风格 B 关键要求）

### 风格 A
- 页眉：机构 logo + 机构名称双语（右上角），小号字体，所有页面均需包含
- 页脚：无（不含页码）

### 风格 B
- **页眉**：报告标题（如"[报告标题]"）居中置顶，楷体小号字体
- **横线**：页眉文字下方细黑色横线，全页宽度
- **页脚**：页码居中或右对齐
- **适用范围**：目录之后的所有内容页
- **目录页**：同样需要页眉+横线；封面页无页眉

## 布局密度与节奏

- **风格 A**：以段落为主，每页 2-4 个段落组，每 3-4 页出现图表
- **风格 B**：密集排版，几乎每页都有图表，文字环绕图表，表格频繁出现
- **行距**：正文 1.5 倍，标题 1.2 倍
- **段间距**：段落后约 0.5em
- **章节分隔**：章节标题前额外垂直间距（约 1em）
- **目录格式**：1 / 1.1 / 1.1.1 三级缩进；点线引导连接标题与页码；页码右对齐

## 品牌/出版方标识

- **风格 A**：红色图标 logo + 双语机构名称置于页眉（右上角）
- **风格 B**：机构 logo 居中于封面页；机构名称出现在页眉文字区域
- 机构名称应在页眉或封面页中始终出现
- 匹配所选参考风格的出版方标识样式
