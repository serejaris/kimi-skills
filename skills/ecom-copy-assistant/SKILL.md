# Product Listing Writer — 电商商品详情页文案知识库

帮助用户撰写专业的电商商品详情页文案，覆盖标题、核心卖点、规格参数、FAQ 四大模块，适配淘宝、京东、亚马逊三大平台的风格规范。

## Quick Start

用户只需说明：
1. **商品信息**：品类、品牌、型号、核心参数
2. **目标平台**：淘宝 / 京东 / 亚马逊（或多平台同时生成）
3. **目标受众**：性别、年龄段、使用场景
4. **语言**：中文 / 英文 / 中英双语
5. **差异化卖点**：与竞品相比的核心优势（可选，由用户提供或从信息中提炼）

示例：
> "帮我写一款无线降噪耳机的商品详情页文案，品牌是SoundMax，型号Pro X3，主打降噪和长续航，要淘宝和亚马逊两个版本。"

## 一、平台风格差异矩阵

### 1.1 三大平台核心规则对比

| 维度 | 淘宝/天猫 | 京东 | 亚马逊 |
|------|----------|------|--------|
| 标题长度 | ≤60字（移动端建议≤30字前置核心词） | ≤75字（建议≤45字） | ≤200字符（建议≤80字符用于前台展示） |
| 标题风格 | 关键词堆叠 + 营销词，热搜词前置 | 品牌+品类+核心参数+场景，结构化清晰 | Brand + Core Feature + Product Type + Key Spec，禁促销词 |
| 卖点形式 | 主图文字 + 详情页长图文 | 商品介绍 + 规格参数表 | Bullet Points（5条，每条≤500字符） |
| 卖点语气 | 感性、煽动、场景化（"拯救你的通勤时光"） | 理性、参数化、权威背书（"XX认证"） | Feature-Benefit 结构，简洁专业 |
| FAQ | 问大家 / 详情页底部 | 商品问答区 | Product Q&A Section |
| 规格表 | 自由格式，常嵌入详情页图片 | 结构化参数表（平台模板） | Product Information 结构化字段 |
| 禁忌词 | "最""第一""绝对"等极限用语（广告法） | 同淘宝 + 虚假宣传类 | "best seller""#1""guaranteed"等夸大词 |
| SEO重点 | 热搜关键词、长尾词、属性词 | 品类词、品牌词、功能词 | Search Terms后台字段 + 标题关键词 |

### 1.2 禁忌词速查表

**中国平台通用禁忌词**（违反广告法）：
- 极限词：最、第一、顶级、极致、绝对、唯一、首个、史无前例
- 虚假承诺：永久、万能、100%、零风险、无副作用
- 权威伪装：国家级、特供、专供（无真实资质时）
- 诱导词：秒杀、清仓（非活动时使用）

**亚马逊禁忌词**：
- 价格/促销相关：cheap, affordable, discount, on sale, free shipping
- 夸大宣传：best, #1, top rated, guaranteed, miracle
- 时效性描述：new, latest（产品更新后失效）
- 卖家主观评价：amazing quality, perfect product
- 竞品提及：better than [Brand X]（标题/Bullet中禁止）

## 二、标题撰写公式

### 2.1 淘宝标题公式

```
[品牌词] + [核心品类词] + [核心卖点×2] + [适用场景/人群] + [修饰词/热搜词]
```

**写作规则**：
- 核心搜索词放前 30 字（移动端截断线）
- 用空格分隔关键词组（系统会自动分词）
- 不加标点符号，不用"的""了"等虚词
- 季节/场景/人群词提升精准流量

**示例**：
```
SoundMax Pro X3 无线蓝牙耳机 主动降噪 50小时续航 头戴式 通勤办公 高音质HIFI
```

### 2.2 京东标题公式

```
[品牌] + [品类] + [型号] + [核心参数×2-3] + [使用场景] + [赠品/服务（如有）]
```

**写作规则**：
- 品牌和品类必须出现在前 15 字
- 用空格分隔语义单元
- 参数用具体数值（"50小时续航"优于"超长续航"）
- 可添加服务承诺（"京东自营""以旧换新"）

**示例**：
```
SoundMax Pro X3 无线蓝牙耳机 主动降噪45dB 50小时续航 头戴式HIFI音质 通勤办公 京东自营
```

### 2.3 亚马逊标题公式

```
[Brand] + [Model] + [Product Type] + [Key Feature 1] + [Key Feature 2] + [Size/Color/Variant]
```

**写作规则**：
- 每个单词首字母大写（Title Case），介词/冠词除外
- 不使用特殊符号（™、®可用）、不全大写
- 品牌名必须位于最前面
- 不加促销信息、价格、主观评价

**示例**：
```
SoundMax Pro X3 Wireless Over-Ear Headphones, Active Noise Cancelling 45dB, 50-Hour Battery Life, Hi-Fi Sound, Foldable Design for Commute and Office
```

## 三、卖点撰写框架

### 3.1 卖点提炼方法论（FAB + SPIN）

每个卖点按 **Feature → Advantage → Benefit** 结构展开：

| 层次 | 定义 | 示例 |
|------|------|------|
| Feature（特征） | 产品具备什么 | 45dB 主动降噪 |
| Advantage（优势） | 比竞品/替代方案好在哪 | 行业领先降噪深度，超越同价位 10dB |
| Benefit（利益） | 给用户带来什么好处 | 地铁/飞机上也能享受纯净音乐，专注不被打扰 |

**卖点优先级排序**：
1. 核心差异化卖点（竞品没有或明显领先的功能）
2. 用户高频痛点对应的功能
3. 品质/安全/认证背书
4. 使用场景扩展
5. 附加价值（赠品、售后、服务）

### 3.2 淘宝卖点写法

**风格**：场景化、情感化、口语化，配合短句和对比。

```
🎵 降噪实力派 | 45dB深度降噪
地铁轰鸣？飞机引擎？统统消失。
搭载旗舰级ANC芯片，降噪深度达45dB，
让你随时随地沉浸在自己的音乐世界。

🔋 超长续航不焦虑 | 50小时畅听
一周只充一次电。
满电50小时连续播放，通勤党再也不用天天找充电线。
快充10分钟 = 听歌3小时，应急超给力。
```

### 3.3 京东卖点写法

**风格**：参数化、技术化、权威感，适当引用认证和测试数据。

```
【45dB 主动降噪】采用自研 ANC 3.0 降噪算法，搭配双馈式麦克风阵列，
实现 45dB 降噪深度。经 SGS 实验室测试认证，有效消减地铁、飞机引擎等
中低频噪音，打造沉浸式聆听环境。

【50 小时超长续航】内置 800mAh 大容量锂电池，满电续航 50 小时
（降噪开启时 35 小时）。支持 Type-C 快充，10 分钟充电可播放 3 小时。
```

### 3.4 亚马逊 Bullet Points 写法

**风格**：Feature-Benefit 并列结构，开头加粗关键词，每条 1-2 句。

```
• INDUSTRY-LEADING NOISE CANCELLATION — Advanced ANC 3.0 technology 
  with dual-feed microphones delivers 45dB noise reduction, effectively 
  blocking subway rumble and airplane engine noise for immersive listening.

• 50-HOUR MARATHON BATTERY — A single charge powers up to 50 hours of 
  playtime (35 hours with ANC on). USB-C fast charging gives you 3 hours 
  of playback from just a 10-minute charge.

• HI-FI AUDIO WITH 40MM DRIVERS — Custom-tuned 40mm dynamic drivers 
  deliver rich bass, clear mids, and sparkling highs. Supports LDAC 
  and AAC codecs for studio-quality wireless audio.

• ALL-DAY COMFORT DESIGN — Memory foam ear cushions with breathable 
  protein leather distribute pressure evenly. Lightweight 250g build 
  and adjustable headband fit all head sizes comfortably.

• SEAMLESS MULTI-DEVICE CONNECTION — Bluetooth 5.3 with multipoint 
  connectivity lets you switch between phone and laptop instantly. 
  Built-in microphone with ENC for crystal-clear calls.
```

## 四、规格参数表模板

### 4.1 通用规格参数模板

| 参数项 | 说明 |
|--------|------|
| 品牌 | 品牌名称 |
| 型号 | 具体型号 |
| 品类 | 商品类目 |
| 颜色/款式 | 可选项列出 |
| 尺寸/重量 | 含包装和单品 |
| 材质/材料 | 主要用材 |
| 核心参数 | 因品类而异的关键技术参数 |
| 兼容性 | 适配设备/系统/标准 |
| 包装清单 | 箱内所有物品 |
| 保修 | 保修期限和范围 |
| 认证 | 获得的质量/安全认证 |
| 产地 | 生产地 |

### 4.2 平台适配注意事项

- **淘宝**：规格信息常以图片形式嵌入详情页，同时填写后台属性字段
- **京东**：必须填写平台规定的结构化规格参数模板，字段不可缺失
- **亚马逊**：填写 Product Information 区域，部分字段为平台必填项，注意单位换算（英寸/磅 for US站）

## 五、FAQ 撰写指南

### 5.1 FAQ 来源与选题

FAQ 应覆盖用户购买决策中的关键疑虑，来源优先级：

1. **竞品评论区高频问题**：看竞品差评和问答区的反复出现问题
2. **品类通用疑虑**：该品类用户普遍关心的核心指标
3. **产品差异化点的延伸问题**：用户对创新功能的理解障碍
4. **售后与使用场景**：退换货、兼容性、使用技巧

### 5.2 FAQ 数量与结构

| 平台 | 建议数量 | 风格 |
|------|---------|------|
| 淘宝 | 8-12 条 | 口语化、买家视角（"这个耳机打游戏延迟高吗？"） |
| 京东 | 6-10 条 | 半正式、技术导向（"该产品是否支持aptX HD编解码？"） |
| 亚马逊 | 5-8 条 | 正式、Feature-Answer结构 |

### 5.3 FAQ 写法示例

**淘宝风格**：

```
Q：降噪效果怎么样？能完全隔绝噪音吗？
A：亲，我们这款耳机降噪深度达到45dB，地铁、办公室空调这些噪音
基本听不到了~不过像人声对话这种高频声音会适当保留，这是为了安全考虑哦。
如果需要完全沉浸可以搭配降噪+音乐一起使用，效果非常棒！

Q：续航真的有50小时吗？
A：50小时是关闭降噪、中等音量下的测试数据。开降噪的话大概35小时左右，
正常通勤使用一周一充完全没问题~而且支持快充，10分钟就能听3小时，
出门前充一会儿就够用了！
```

**京东风格**：

```
Q：本产品的降噪等级和降噪深度具体是多少？
A：本产品搭载 ANC 3.0 主动降噪技术，降噪深度为 45dB，经 SGS 实验室
环境测试认证。可有效消减 50-1000Hz 频段的环境噪音（如交通噪音、
空调运转声等），高频人声段保留约 30% 以确保安全通行需要。

Q：电池容量和实际续航时间分别是多少？
A：内置 800mAh 锂聚合物电池。续航时间：纯音乐模式约 50 小时，
ANC 开启模式约 35 小时（测试条件：音量 50%，25°C 室温）。
支持 USB-C 快充，充电 10 分钟可使用约 3 小时。
```

**亚马逊风格**：

```
Q: How effective is the noise cancellation?
A: The Pro X3 features Active Noise Cancellation with 45dB noise reduction,
verified by SGS lab testing. It effectively reduces low-to-mid frequency 
noise such as airplane engines, subway rumble, and office HVAC. Ambient 
sound mode is available for situational awareness when needed.

Q: What is the actual battery life with ANC turned on?
A: With ANC enabled at 50% volume, the battery lasts approximately 
35 hours. With ANC off, battery life extends to approximately 50 hours. 
USB-C fast charging provides 3 hours of playback from a 10-minute charge.
```

## 六、SEO 关键词策略

### 6.1 关键词分层

| 层级 | 定义 | 示例（无线耳机） | 用途 |
|------|------|----------------|------|
| 核心词 | 品类大词 | 无线耳机、蓝牙耳机 | 标题必含 |
| 属性词 | 功能/参数描述 | 降噪、长续航、头戴式 | 标题 + 卖点 |
| 场景词 | 使用场景 | 通勤、办公、运动 | 标题 + 详情页 |
| 长尾词 | 细分需求 | 飞机降噪耳机、打游戏低延迟耳机 | 标题尾部 + 后台搜索词 |
| 竞品词 | 竞品品牌/型号 | — | 仅限亚马逊后台 Search Terms |

### 6.2 亚马逊 Search Terms 填写规则

- 总长度 ≤250字节（含空格）
- 不要重复标题中已出现的词
- 用空格分隔，不要用逗号或分号
- 不要包含品牌名（已在标题中）
- 可包含拼写变体、同义词、西班牙语/法语常见搜索词（对应站点）

## 七、Agent 行为指南

当用户要求撰写电商商品文案时，按以下流程执行：

### 步骤 1：信息收集

确认以下关键信息（缺失项用占位符标注并提醒用户补充）：
- 商品基本信息：品类、品牌、型号
- 核心参数/卖点：至少 3 个差异化特征
- 目标平台：淘宝 / 京东 / 亚马逊（可多选）
- 目标受众：人群画像和使用场景
- 语言：中文 / 英文 / 中英双语
- 竞品参考（可选）：用户指定的竞品或对标产品

### 步骤 2：卖点提炼与排序

1. 从用户提供的信息中提炼 5-8 个卖点
2. 按 FAB 结构展开每个卖点
3. 按优先级排序（差异化 > 痛点解决 > 品质背书 > 场景扩展）

### 步骤 3：按平台生成文案

对每个目标平台，依次输出：

```
━━━━━━━━━━━ [平台名称] ━━━━━━━━━━━

📌 标题
[按该平台标题公式生成]

🎯 卖点（Bullet Points / 详情文案）
[按该平台风格生成 5 条核心卖点]

📋 规格参数表
[按该平台格式输出]

❓ FAQ（5-8 条）
[按该平台风格生成]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 步骤 4：合规检查

生成后自动检查：
- [ ] 是否包含禁忌词/极限用语
- [ ] 标题是否超出平台字数限制
- [ ] 参数数值是否有依据（不可凭空编造性能数据）
- [ ] Bullet Points 是否超出字符限制（亚马逊每条≤500字符）
- [ ] 是否有夸大宣传或虚假承诺

### 步骤 5：输出与说明

- 完整输出所有平台的文案
- 标注需要用户确认或补充的占位信息
- 说明卖点排序逻辑和关键用词选择理由
- 如用户未提供某些参数数据，明确标注"[需确认]"而非自行编造

## 八、品类适配要点

不同品类的文案侧重点不同，以下为常见品类的核心关注点：

| 品类 | 标题关键词侧重 | 卖点侧重 | FAQ 高频问题 |
|------|---------------|----------|-------------|
| 3C数码 | 型号、核心参数、兼容性 | 性能数据、续航、认证 | 兼容性、真实续航、保修 |
| 服装鞋包 | 风格、材质、季节、场景 | 面料手感、版型、穿搭场景 | 尺码选择、洗涤保养、色差 |
| 美妆护肤 | 功效、成分、肤质适用 | 核心成分、使用效果、安全性 | 敏感肌能用吗、使用频率、保质期 |
| 食品饮料 | 口味、规格、产地、保质期 | 原料品质、口感描述、健康价值 | 保质期、过敏原、储存方式 |
| 家居家电 | 尺寸、功率、能效等级 | 节能数据、使用便利性、安全认证 | 安装方式、能耗、售后服务 |
| 母婴用品 | 年龄段、安全认证、材质 | 安全无毒、权威检测、使用场景 | 适用年龄、安全认证、清洗方式 |

## 九、文案质量评估标准

生成的文案应满足以下标准：

| 维度 | 合格标准 |
|------|---------|
| 平台合规 | 无禁忌词、字数符合平台限制、格式正确 |
| 信息准确 | 参数数据有据可查、不编造性能指标 |
| 卖点清晰 | 每个卖点 FAB 完整、差异化明确 |
| 受众匹配 | 语气风格契合目标用户画像 |
| SEO 友好 | 核心关键词自然融入标题和卖点 |
| 可操作性 | 占位信息标注清晰、用户可直接修改使用 |
