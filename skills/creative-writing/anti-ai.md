# AI 八股检查表（中文分类大全 + 跨语言硬性禁项）

> **用法**：起草前通读一遍、**自审时**（review.md Pass 2）逐项扫描，命中即按"→"后的方式修改。八股分两种性质：
> - **模式类**（§一～§五）：出现一次就是问题的词、句式、手法——逐项人工扫描。
> - **频率类**（§六）：单处无害、密度超标才是八股——**必须程序实测**（`scripts/scan_anti_ai.py` 或等效 ipython 统计），肉眼会在起草中将其正常化。
>
> **适用性**：条目是叙事散文的默认值，不是铁律。当某表达是人物声音、体裁或风格的**有意选择**时可以保留——但必须是有意识的少数例外，且说得出理由，不是惯性输出。

## 零、跨语言硬性禁项（任何语言的创作正文中一次都不得出现）

- **引用/工具痕迹**：`oaicite`、`turn0search`、`[^n^]` 式引用标记、工具调用残留。
- **知识截止语**：as of my last update / 截至我的知识更新 等。
- **聊天机器人口癖**："I hope this helps!"、"Certainly!"、「希望对你有帮助」、「如果还有其他需要…」。
- **Markdown 泄漏**：正文中的行首 `#`、`**`、反引号——**包括直接写在对话框里的正文**。

## 英文高危词表（English high-signal AI vocabulary — 英文创作时逐项扫描）

- 名词/形容词：tapestry, delve, pivotal, foster, underscore, intricate, multifaceted, myriad, beacon, vibrant, breathtaking; "stands as a testament to"。
- **系动词回避**（copula-avoidance）："serves as" / "boasts" 用在 "is" / "has" 更自然的地方 → 直接用 is/has。
- 结构套路："It's not just X, it's Y"、rule-of-three 排比、每段等长 → 参照§二、§六同类中文条目处理。

## 一、词语与短语类（滥用的词与套话）

**翻译腔与框架词** — 直接写，删掉框架：
- 名为……的……/被称为/叫做 → 直接指明事物
- 令人……（-able 直译）→ 具体感受（令人窒息 → 喘不过气）
- 不容置疑/不容分说/不容置喙 → 删去
- 成功地/进一步/更好地 → 直接写结果或后置补语（建设得更好）
- 作为……/通过……/关于…… → 据语境改写（为人师表/贵为天子）或直接写方式、直接说事
- 最……之一 → 直接判断（是伟大的科学家）
- 在……的过程中/在……的基础上/从……的角度来看/对于……来说/随着……的发展 → 直接写动作/前提/观点/变化
- 有（着）……的特点/具有……的……性 → 直接写属性（这方案靠谱）
- 弱动词+名词（作出决定/进行调查/予以表彰）→ 直接用动词（决定/调查/表彰）
- 关联词冗余（因为…所以/虽然…但是/即使…也/如果…的话/只要…就/由于…因此/一方面…另一方面/不仅…而且/既…又）→ 中文常可省略关联词、条件前置或拆为两句
- 如此……以至于…… → 拆句重构
- 肉眼可见地/不可否认地/显而易见地 → 删副词，直接写画面

**空话套话** — 删除：
- 所谓/从某种意义上说/不可否认的是/毫无疑问的是/众所周知 → 删除或直接写
- 总而言之/综上所述/总的来说/值得一提的是/特别需要指出的是 → 删除
- 在这个……的时代/背景下、在这个问题上、在这种情况下 → 删除

**模糊量词与定性框架** — 删框架，写具体：
- 一丝/一抹/一阵/一缕/一股+抽象名词 → 删量词框架，直接写具体状态
- 带着一丝不易察觉的……/不易察觉/几不可察 → 直接写观察到的细节
- 难以言喻的……/无法名状的…… → 转为具体身体反应或想法
- 有（一种）……的感觉/情绪 → 删框架，直接写感受
- 极其/极为/极度/无比 → 用具体画面让读者自行判断程度
- 取而代之 → 直接描写变化前后的状态
- 引以为傲的…… → 祖传手艺/看家本领/压箱底的本事
- 声音/语气带着……/透着……/夹杂着……（状态容器）→ 情态副词+发声动词，或删除

**身体与感官套路词**：
- 嘴角勾起弧度/上扬/微扬/带着一抹…… → 直接写表情或删去
- 心跳漏了一拍/心提了起来/脑中一片空白/心沉入谷底/心如刀割/心中掀起惊涛骇浪 → 删去，写外部可观察反应或具体行为
- 喉结滚动/喉咙滚了一下 → 改为「咽了一下」；青筋浮起/身子微微一顿 → 少用
- 胸腔震动/胸腔起伏/生理性的…… → 直接写表情动作或情绪
- 内脏反应（胃翻/反胃/恶心/肠绞/酸水上涌）→ 写外部可观察结果（捂嘴、弯腰、干呕的动作）
- 解剖词降级：耻骨/肋骨/肩胛骨/眉骨/指骨/骨节分明/指节泛白/粗糙的指腹/薄茧 → 肩膀、手、手指、眉眼等平实统称
- 沉声/低吼/低喝 → 情态副词+发声动词（低声说）
- 冷冷地/冷笑/冷哼（以"冷"为核心贴标）→ 直接写动作和表情
- 戏谑/狡黠/狰狞/侵略性（形容词贴标签）→ 用动作和台词让人物自己呈现
- 四肢百骸/虚无缥缈/画龙点睛/吐气如兰/眼神空洞 → 删去

**称谓与油腻词**：
- 这女人/这丫头/这姑娘/金主/甲方/房东 → 用名字、「你」或符合情感距离的自然称谓
- 老子（自称）→ 说「我」（除非人设如此）
- 征服/支配/占有/沉沦/屈辱/猎手/猎物/狩猎/共犯 → 删除，直接写人物关系与行为
- 身体比脑子诚实/脸红心跳 → 删去

**学术与议题词入叙事** — 用日常人话，或不入正文：
- 张力/解构/规训/赋权/凝视/共鸣/联结/对立 → 日常表达（直接写紧张感如何产生）
- 父权/男性凝视/厌女/女性主义/社会达尔文主义/弱肉强食/优胜劣汰/政治正确/身份政治/结构性压迫 → 议题标签不进入叙事正文

## 二、句式类（结构陷阱）

- 不是A而是B/不是…是…/并非…而是/没有…只有/没有…而是 → 直接写B（偶发可容，禁止成为惯性句式——频率见§六）
- 是+状态变化/是……的（判断词滥用）→ 删「是」，直接陈述
- 被动句滥用/像是被什么东西+动词 → 化被动为主动
- 长定语前置（定语从句翻译腔）→ 拆短句，流水句推进
- 形容词+的+名词（冗长定语）→ 能删则删（红花/利刃）；副词+地+动词 → 能删则删（慢慢走/大喊）
- 一个/一种/一丝+名词 → 非强调数量则删
- 程度从句 A到B（快到轮廓拉成残影）/……得只剩/……得只有 → 拆句，直接写具体状态
- 前句出现核心词、后句重复兑现同一词义（他很熟悉，熟悉到……）→ 删程度连接，直接接续
- 时间框架（就在……的瞬间/那一刻/那一刹那）→ 删去，用动作时序表达
- 事件主语因果（[名词/从句]+让我/令我/使我+反应）→ 删因果框架，直接写反应
- 比较级翻译腔（比+任何/所有/每一个……都……）→ 直接写感受强度
- 存在引出（然后是/接着是/剩下的是+名词）→ 改用动词
- 物品主语（[无生命物]+遮住/笼罩/把+人）→ 改以人或被影响者为主语
- 视线位移（目光/视线/眼神+越过/扫过/落在/投向）→ 人做主语的「看着」「盯着」
- 叙述者自我纠正（不对，X这个词太Y了/与其说是X不如说是Y）→ 一开始就用最准确的词
- 无效否定（并未/没有/几乎没+动作，用于铺垫）→ 写实际状态
- 否定铺垫（不仅……/甚至顾不上……）→ 删除否定框架
- 解释性同位语（[名词]。是[解释]。）→ 融入前句或删除
- 冗余并列（或者/以及+抽象物）→ 后半赘余则删
- 同一人连续动作主语切换为身体零件（身子/双手/嘴唇+动词）→ 全程一个主语，分句省略主语流水推进
- ≤3 秒内的动作序列拆成多个独立短句 → 整合为一个完整句
- 依然/仍然/还是/跟之前一样/和往常一样（常态对比）→ 删除，直接写当前状态
- 冒号滥用（对话引出冒号/叙述冒号）→ 改逗号

## 三、描写手法类（叙事层面的套路）

**比喻**（配合§六的密度阈值；好比喻新鲜、准确、不抢戏）：
- 意义盖章式比喻（画面已呈现后追加「像……」做情绪定性/意义提炼）→ 删除
- 解释性比喻（用比喻解释动作的性质程度：熟练得像……）→ 删除
- 比喻+效果/行为（他的话像刀扎进她的心/像是要暴起伤人）→ 直接写效果、写动作
- 陈词喻体（手术刀/超级计算机/破布娃娃/猎豹/闪电/溺水抓浮木/达摩克利斯之剑/石子/雕像/珍宝/棋子/炸弹/救命稻草/烙印/面具/羽毛/热油/困兽/小兽/针/刺/弓弦/锤子/钉子/一叶扁舟）→ 删除或换新鲜准确的喻体
- 微比喻容器（心湖/心田/脑海/心弦）→ 直接写感受
- 通感贴标（眼神锐利/声音甜美/冰冷的话语）→ 直笔白描
- 类比说理（这就好比……）→ 用逻辑推演

**镜头与动作**：
- 扫描仪式环境穷举（逐个罗列环境细节）→ 每次场景交互至多 1–2 个关键细节
- 流水账式动作记录/慢动作分解（单个动作拆成子阶段）→ 结果 + 一个最突出的视觉特征
- 身体零件/声音/语气做主语（喉咙滚了一下/声音传来/语气里透着）→ 人做主语；声音属性描述（嗓音压得又平又稳）→ 情态副词+发声动词（平平稳稳地说）
- 声音物理化（从……飘出/砸在……上/回荡）、声音传递过程 → 删除
- 虚空/习惯性动作（对不存在的东西做动作）→ 动作必须符合角色当前肢体位置与场景物件

**叙述者越位**（画面已呈现，叙述者还要替读者下结论）：
- 事后定性（那是她的身体在……/那是一种……）→ 删除
- 话术分析（他说的X其实意味着Y）/动机推断（他在提醒她/她故意岔开话题）/处境总结（她不让我走/他们在排挤我）→ 删除，只呈现画面
- 抽象化行为评价（她在跟自己较劲/他在忍/她在撑/他在演）→ 用具体动作替代
- 极短定性句（是在审/这是威胁/她在试探/他在等）→ 删除，让读者自行理解
- 眼神情绪解读（眼底闪过/透着[情绪词]）、状态介词外挂（带着/透着/显得+状态）→ 删除
- 认知/感知声明引出（我想起/我记得/我看清了/我听见了/我闻到了）→ 记忆与感知以画面碎片直接嵌入叙事流
- 过度解读/升华拔高/引申寓意/主题宣讲式结尾（这就是……的悲剧/一切都会好起来的）→ 删除，收在具体画面上

**对话周边**：
- 台词回声（台词后第一个叙述句复用该台词实词）/台词后解释潜台词 → 删除，接物理动作
- 对话标签三重堆叠 → 合并或删除
- 台词书面化 → 台词必须短、口语化、有语气词
- 他的话/她的话/这句话（台词余波框架）→ 直接接物理动作

**节奏**（与 content.md 的节奏原则配合——短句打点是好手法，这里限制的是**连续机械碎片**）：
- 连续 3 句以上 ≤12 字的句号句 → 合并为复合句
- 无动词名词碎片（「女人。」「黑暗。」）→ 融入相邻句或加动词
- 同一句内 2 个以上 ≤4 字微碎片 → 合并
- 内心活动逐拍拆成连续独立短句 → 合并为复合句

**讲史/说书腔**（历史、设定向叙事高发——背景通过场景与人物视角给出，不用 lecturer 视角）：
- 元话语（让我们回到……/要讲X，先得讲讲Y/这个问题千百年来没有人……/可事情真的这么简单吗？/翻译成白话就是）→ 删除
- 精确数值/解剖术语/物理计算入叙事 → 替换为主观感受
- 关键时刻喊出"就是现在"然后开大招 → 禁止

## 四、角色塑造类（人设套路）

> **适用性**：本节针对的是**无意识默认值**——不加思考就给所有角色装上的同一套零件。反派可以刻薄，权谋角色可以居高临下，黑暗题材可以残酷：**由人设和题材驱动的"恶"是合法塑造**。要消除的是换到任何角色身上都成立的惯性预设。

- 男性角色默认强势/强制/占有姿态 → 允许脆弱、哭泣、委屈、害羞、紧张；强势必须出自人设而非默认
- 外观惯性预设（洗得发白的白衬衫+皂角味/粗硬胡茬/手指修长骨节分明有薄茧/默认抽烟）→ 无明确设定不得套用
- 角色全知（读心术/预知未暴露的计划）→ 引号外的心理活动不可知；推理必须包含误判可能
- 角色使用未获得的信息 → 只用亲眼见过、被告知过、或能推理出的信息
- 角色一成不变（永远游刃有余/冷静从容）→ 展现失控、挣扎、犹豫、逻辑断层
- 行为未被人设驱动（放任何角色身上都成立）→ 作废重写

## 五、流程类（写作过程的套路）

- 全局重写（发现错误后整段推倒）→ 最小修改原则：只改违规部分
- 跳过审查直接输出 → 成稿必须过 review.md 的扫描流程（模式类人工扫 + 频率类程序测）
- 段落不含新信息（删掉后上下文仍通顺）→ 废段，作废重写
- 动作不连贯（未确认角色当前肢体位置、朝向、姿势）→ 写动作前确认肢体状态

## 六、频率阈值（程序扫描项）

> 以下各项**单处出现不算错，密度超标才是八股**。自审时对成稿文件实测（`scripts/scan_anti_ai.py <file>`）；**续写/改编任务以原作实测值为基准**（保持在原作频率的 ±50% 内），下表为无原作参照时的默认上限。

| 项 | 默认上限 | 超标处理 |
|---|---|---|
| 明喻引导词（像/像是/仿佛/如同/宛如/犹如/好似） | ≤3 处/千字 | 先删意义盖章式与陈词喻体，保留新鲜准确的 |
| 破折号（——） | ≤2 处/千字 | 改逗号/句号/另起句 |
| tic 词（缓缓/微微/轻轻/淡淡/嘴角） | 合计 ≤5 处/千字 | 删除或换具体动作 |
| 「不是X而是Y」族 | ≤1 处/千字 | 直接写B |
| hedging 连用（也许是…也许是…/或许…或许…） | 全文 ≤2 组 | 叙述者下判断，或干脆留白 |
| 对偶/排比 ≥3 连 | 全文 ≤2 组 | 拆散重写 |
| 四字格律情绪词（心潮澎湃/热血沸腾/百感交集） | 全文 ≤2 处 | 身体反应替代 |
| Markdown 痕迹（行首 `#`、`**`、反引号） | 正文中 0 | 必须清除 |
