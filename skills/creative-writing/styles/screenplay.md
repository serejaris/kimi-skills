# Screenplay / Storyboard Style

Film/TV scripts and 分镜. The script conveys only what can be **seen and heard** — no interior prose.

**Format is not language.** `FADE IN:` / `INT./EXT.` are layout conventions — a 中文 brief gets a 中文剧本 in FADE IN format (scene headings may mix English markers with Chinese text). Never switch the writing language because the requested format looks Western.

## Voice and Tone
- Objective present-tense action lines. No novelistic interiority ("she remembers…", "he feels…") — externalize it into behavior, image, and dialogue.
- Lean and visual. Every line is a shot or a sound.

## Structure & Format
- Standard screenplay format unless the user requests otherwise:
  - `FADE IN:` to open.
  - **Scene heading**: `INT./EXT. LOCATION — DAY/NIGHT` (中文剧本可用「场N 地点 内/外 日/夜」).
  - **Action lines**: present tense, concise, one action beat per short paragraph.
  - **Character cue** (centered/caps) above dialogue; **(parenthetical)** only when necessary; **dialogue** below.
  - Transitions (`CUT TO:`, `DISSOLVE TO:`) sparingly.
- For 分镜/storyboard: per shot give shot size (远/全/中/近/特), camera move, action, and any dialogue/SFX.
- Honor the user's requested format exactly (e.g., "fade in 格式 中文版").

## Language Patterns
- Dialogue does the character work — distinct voices, subtext, interruption, what's not said. Trim to the sharpest version.
- Action lines: concrete, filmable verbs; no camera directions unless the user wants them; no unfilmable description ("the saddest man in the city").
- White space is structural — short paragraphs, fast read.

## Prohibited Patterns
- On-the-nose dialogue where characters announce their feelings/plot.
- Wall-of-text action blocks; novelistic interiority; describing thoughts that can't be filmed.
- Inconsistent or non-standard format when a standard was requested.
- Generic stage directions ("they talk for a while").
