# Structure Contract — Investment Memo

Two reference styles: narrative/thematic investor letter and structured VC deal memo.

## Document Types

### Type A: Thematic / Macro Memo (Narrative Investor Letter Style) — PRIMARY

Narrative, essay-style, reflective. Typically 8-20+ pages. Structure follows the logical flow of the argument. This is the default style.

**Opening**: No executive summary. The first body paragraph sets the context. Begin with "When I was...", "I recently...", or similar personal framing that explains why the memo is being written now.

```
MEMO HEADER (page 1, top)
  Memo to:     [Clients / Partners / Investment Committee]
  From:        [Author]
  Re:          [Subject / Theme]
  Date:        [Date]
  ────────────────────────────────────────────────────────

OPENING / CONTEXT (1-2 pages)
  No heading needed — the Re: line serves this purpose
  └─ Personal framing: why writing this now, what question is addressed
  └─ First-person narrative: "I've been thinking about...", "I want to share..."

BACKGROUND / EXPLAINER (2-5 pages)
  Section heading: Underlined text on its own line
  ├─ What is [the topic] — definitions and key concepts
  ├─ How did we get here — historical context and timeline
  ├─ Key developments explained accessibly
  └─ Indented excerpts from third-party sources (italic, no borders)

ANALYSIS / DISCUSSION (3-8 pages)
  Section headings: Underlined text
  ├─ Arguments build progressively through sections
  ├─ Each section tackles one facet of the question
  ├─ Use analogies and historical precedents
  ├─ Bold key sentences that anchor the reader's attention
  ├─ Bullet lists for enumerated points (numbered only if sequential/temporal)
  └─ Address counterarguments fairly

IMPLICATIONS FOR INVESTING (2-4 pages)
  Section heading: Underlined text
  ├─ What this means for portfolio positioning
  ├─ Asset class / sector implications
  ├─ Risks and opportunities
  └─ Timing considerations

BOTTOM LINE / CONCLUSION (1 page)
  Section heading: Underlined text (or "Conclusion")
  ├─ Summary judgment in plain language
  ├─ Key takeaways as bullet points (optional)
  ├─ Actionable guidance where applicable
  └─ Humility / uncertainty acknowledgment

POST-SCRIPT (optional, 0.5-1 page)
  "P.S." heading underlined
  ├─ Additional thoughts that emerged after main writing
  ├─ Related topics for future memos
  └─ Follow-up observations

LEGAL DISCLAIMERS (1 page)
  ────────────────────────────────────────────────────────
  *Legal Information and Disclosures*  (italic, underlined)

  [Italic disclaimer text: views subject to change, past performance
   not indicative of future results, not an offer, sources disclaimer,
   reproduction restrictions]
```

### Type B: Venture / Deal Memo (Top-Tier VC Style)

Concise, structured, decision-oriented. Typically 5-10 pages. Use when analyzing a specific investment opportunity.

```
MEMO HEADER (page 1, top)
  To:     [Investment Committee / Partners]
  From:   [Author / Deal Lead]
  Re:     [Company Name — Seed/Series X Investment]
  Date:   [Date]
  ────────────────────────────────────────────────────────

EXECUTIVE SUMMARY / INVESTMENT THESIS (1 paragraph)
  No separate heading needed
  ├─ One-sentence thesis: what the company does, why it wins
  ├─ Key metrics: stage, amount, valuation, ownership
  └─ Recommendation: invest / pass / conditional

COMPANY OVERVIEW
  Section heading: Bold text
  ├─ Company purpose (1 sentence, italic optional)
  ├─ What the company does (2-3 paragraphs)
  ├─ Founding story / team background
  └─ Stage of development

PROBLEM
  Section heading: Bold text
  ├─ Pain point being solved
  ├─ Evidence (data, anecdotes)
  └─ Why existing solutions fail
  Use dash bullets for enumerated problem points

SOLUTION / PRODUCT
  Section heading: Bold text
  ├─ Product description
  ├─ Key differentiators
  └─ Technology / moat

MARKET OPPORTUNITY
  Section heading: Bold text
  ├─ TAM / SAM / SOM (inline text, avoid tables)
  ├─ Growth drivers
  └─ Timing: why now?

BUSINESS MODEL & UNIT ECONOMICS
  Section heading: Bold text
  ├─ Revenue model
  ├─ Unit economics (LTV/CAC, payback) in bullet form
  └─ Path to profitability

COMPETITIVE LANDSCAPE
  Section heading: Bold text
  ├─ Direct competitors (bullet list: name — brief assessment)
  ├─ Indirect competitors
  └─ Why this company wins

TEAM & TALENT
  Section heading: Bold text
  ├─ Founders (background, prior success)
  ├─ Key hires
  └─ Gaps / hiring plan

DEAL TERMS
  Section heading: Bold text
  ├─ Investment amount and structure
  ├─ Valuation
  ├─ Ownership percentage
  └─ Use of proceeds

RISKS
  Section heading: Bold text
  ├─ Key risk factors as bullet points
  ├─ Severity assessment inline
  └─ Mitigation strategies

RECOMMENDATION
  Section heading: Bold text
  ├─ Invest / Pass / Conditional — state clearly
  ├─ Rationale in 3-5 bullet points
  └─ Next steps

EXHIBITS (inline or appendix)
  ├─ Simple line charts only (no styled charts)
  ├─ Exhibit label: "Exhibit 1: [Description]"
  └─ Sequential numbering across all exhibits
```

## Section Hierarchy

- **H1 (Major sections)**: Underlined text on its own line (narrative) or bold text (structured)
- **H2 (Sub-sections)**: Bold text within the flow (use sparingly)
- **H3**: Not typically needed in memos

Section headings do NOT use numbering (no "1. Introduction", "2. Analysis", etc.). Narrative memos may have numbered bullet points within sections but never numbered sections themselves.

## Exhibit Numbering

Exhibits are numbered sequentially across the entire document: **Exhibit 1**, **Exhibit 2**, etc. Reference in text: "(see Exhibit 3)" or "as shown in Exhibit 3."

## Page Breaks & Section Breaks

- **No page breaks between body sections** — continuous text flow is the standard
- **Before exhibits appendix**: Optional page break if exhibits are grouped at the end
- **Legal disclaimers**: Start on a new page if they would otherwise begin mid-page; separate from body by a horizontal rule

## Header/Footer Rules

| Element | First Page | Body Pages |
|---------|-----------|-----------|
| Page number | Arabic numeral centered in footer | Arabic numeral centered in footer |
| Running header | NONE | NONE |
| Confidentiality | Bottom-left in footer (if applicable) | Bottom-left in footer |
| Copyright | Bottom-left in footer | Bottom-left in footer |
| All Rights Reserved | Bottom-right in footer | Bottom-right in footer |

## Front Matter

- NO cover page
- NO table of contents
- NO abstract or executive summary (except for deal memos)
- The memo header block serves as the document identification

## Back Matter

- **Exhibits**: Inline or brief appendix with simple charts
- **Disclaimer page**: Legal text in italic, preceded by horizontal rule, optional "Legal Information and Disclosures" heading
- **No separate sources/references page** — cite sources inline or in footnotes

## Section Ordering Flexibility

For deal memos, the specific order of sections can be adapted based on:
- Deal urgency: Put recommendation and deal terms early if IC meeting is imminent
- Thesis strength: Lead with the strongest argument first
- User preference: Follow any specific section order the user requests
