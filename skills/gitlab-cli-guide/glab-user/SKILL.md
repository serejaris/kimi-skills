# glab user

## 概述

```

  Interact with a GitLab user account.                                                                                  
         
  USAGE  
         
    glab user <command> [command] [--flags]  
            
  COMMANDS  
            
    events [--flags]  View user events.
         
  FLAGS  
         
    -h --help         Show help for this command.
```

## 快速入门

```bash
glab user --help
```
