# 结构规范 — 投资备忘录

两种参考风格：叙事型/主题型投资人信函和结构化风投交易备忘录。

## 文档类型

### A类：主题/宏观型备忘录（叙事型投资人信函风格）——主要类型

叙事型、散文风格、反思性。通常8-20页以上。结构跟随论证的逻辑展开。这是默认风格。

**开头**：无执行摘要。正文第一段直接交代背景。以"在我……的时候"、"我最近……"或类似的个人化表述开头，解释为何此时撰写这份备忘录。

```
备忘录头部信息（第1页，顶部）
  Memo to:     [客户 / 合伙人 / 投委会]
  From:        [作者]
  Re:          [主题]
  Date:        [日期]
  ────────────────────────────────────────────────────────

开头 / 背景交代（1-2页）
  无需标题 — Re: 行已起到此作用
  └─ 个人化引入：为什么现在写这篇，要探讨什么问题
  └─ 第一人称叙述："我一直在思考……"、"我想分享……"

背景 / 解读（2-5页）
  章节标题：下划线文字独占一行
  ├─ [主题]是什么 — 定义与核心概念
  ├─ 我们如何走到今天 — 历史背景与时间线
  ├─ 以通俗易懂的方式解释关键进展
  └─ 缩进引用第三方资料（斜体，无边框）

分析 / 讨论（3-8页）
  章节标题：下划线文字
  ├─ 论证在各章节间逐步推进
  ├─ 每个章节聚焦问题的一个方面
  ├─ 使用类比和历史先例
  ├─ 加粗关键句以锚定读者注意力
  ├─ 枚举要点使用项目符号列表（仅顺序/时间性内容才编号）
  └─ 公正地回应反方论点

投资启示（2-4页）
  章节标题：下划线文字
  ├─ 这对投资组合配置意味着什么
  ├─ 资产类别/行业的影响
  ├─ 风险与机会
  └─ 时机考量

核心结论（1页）
  章节标题：下划线文字（或"结论"）
  ├─ 用通俗语言给出总结性判断
  ├─ 关键要点以项目符号呈现（可选）
  ├─ 在适当情况下给出可操作的建议
  └─ 表达谦逊 / 承认不确定性

后记（可选，0.5-1页）
  "P.S."标题加下划线
  ├─ 主体写作完成后产生的补充想法
  ├─ 未来备忘录可能探讨的相关话题
  └─ 后续观察

法律免责声明（1页）
  ────────────────────────────────────────────────────────
  *法律信息与披露*（斜体，加下划线）

  [斜体免责声明文字：观点可能变更、过往业绩不代表
   未来表现、非要约邀请、信息来源说明、
   转载限制]
```

### B类：风投/交易型备忘录（顶级风投风格）

简洁、结构化、决策导向。通常5-10页。用于分析具体的投资机会。

```
备忘录头部信息（第1页，顶部）
  To:     [投委会 / 合伙人]
  From:   [作者 / 项目负责人]
  Re:     [公司名称 — 种子轮/X轮投资]
  Date:   [日期]
  ────────────────────────────────────────────────────────

执行摘要 / 投资论点（1段）
  无需单独标题
  ├─ 一句话论点：公司做什么，为什么能赢
  ├─ 关键指标：阶段、金额、估值、持股比例
  └─ 建议：投资 / 放弃 / 有条件投资

公司概况
  章节标题：加粗文字
  ├─ 公司使命（1句话，可用斜体）
  ├─ 公司业务（2-3段）
  ├─ 创始故事 / 团队背景
  └─ 发展阶段

核心问题
  章节标题：加粗文字
  ├─ 要解决的痛点
  ├─ 证据（数据、案例）
  └─ 现有方案为何不够好
  枚举问题要点使用短横线列表

解决方案 / 产品
  章节标题：加粗文字
  ├─ 产品描述
  ├─ 关键差异化优势
  └─ 技术 / 壁垒

市场机会
  章节标题：加粗文字
  ├─ TAM / SAM / SOM（行内文字，避免表格）
  ├─ 增长驱动力
  └─ 时机：为什么是现在？

商业模式与单位经济
  章节标题：加粗文字
  ├─ 收入模式
  ├─ 单位经济（LTV/CAC、回收周期）以要点形式呈现
  └─ 盈利路径

竞争格局
  章节标题：加粗文字
  ├─ 直接竞争对手（要点列表：名称 — 简要评价）
  ├─ 间接竞争对手
  └─ 为什么这家公司能赢

团队与人才
  章节标题：加粗文字
  ├─ 创始人（背景、过往成绩）
  ├─ 核心人员
  └─ 能力缺口 / 招聘计划

交易条款
  章节标题：加粗文字
  ├─ 投资金额与结构
  ├─ 估值
  ├─ 持股比例
  └─ 资金用途

风险
  章节标题：加粗文字
  ├─ 关键风险因素以要点列出
  ├─ 行内标注严重程度
  └─ 缓释措施

投资建议
  章节标题：加粗文字
  ├─ 投资 / 放弃 / 有条件 — 明确表态
  ├─ 理由以3-5个要点阐述
  └─ 后续步骤

图表（嵌入正文或附录）
  ├─ 仅使用简单折线图（不用花哨的图表样式）
  ├─ 图表标签："图表 1：[描述]"
  └─ 全文顺序编号
```

## 章节层级

- **一级标题（主要章节）**：下划线文字独占一行（叙事型）或加粗文字（结构化型）
- **二级标题（子章节）**：正文中使用加粗文字（谨慎使用）
- **三级标题**：备忘录中通常不需要

章节标题不使用编号（不用"1. 简介"、"2. 分析"等）。叙事型备忘录在章节内可使用编号的要点列表，但章节本身绝不编号。

## 图表编号

图表在全文范围内顺序编号：**图表 1**、**图表 2**，依此类推。正文中引用："（见图表 3）"或"如图表 3 所示"。

## 分页与分节

- **正文章节间不分页**——文字连续流转是标准做法
- **图表附录前**：如图表集中放在文末，可选择性分页
- **法律免责声明**：如果否则会从页面中间开始，则另起一页；以水平分割线与正文分隔

## 页眉/页脚规则

| 元素 | 第一页 | 正文页 |
|------|--------|--------|
| 页码 | 阿拉伯数字居中于页脚 | 阿拉伯数字居中于页脚 |
| 页眉 | 无 | 无 |
| 保密标记 | 页脚左下角（如适用） | 页脚左下角 |
| 版权声明 | 页脚左下角 | 页脚左下角 |
| 版权所有 | 页脚右下角 | 页脚右下角 |

## 前置页

- 无封面页
- 无目录
- 无摘要或执行摘要（交易型备忘录除外）
- 备忘录头部信息块即为文档标识

## 后附页

- **图表**：嵌入正文或简短附录配以简单图表
- **免责声明页**：斜体法律文字，前以水平分割线分隔，可选"法律信息与披露"标题
- **不设单独的参考文献页**——在正文中或脚注中标注来源

## 章节排列的灵活性

交易型备忘录的章节顺序可根据以下情况调整：
- 交易紧迫度：如投委会会议临近，将投资建议和交易条款前置
- 论点强度：最有力的论据先行
- 用户偏好：按照用户要求的特定顺序排列
