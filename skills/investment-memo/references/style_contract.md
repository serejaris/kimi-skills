# 样式规范 — 投资备忘录

两种参考风格：顶级信用/价值投资者的叙事型投资人信函和顶级风投机构的结构化交易备忘录。

## 核心原则：朴素的、以文字为核心的文档

投资备忘录是**内容载体，而非设计作品**。两份参考备忘录都刻意采用极简风格：白纸黑字，无装饰元素，无颜色，无样式框线。专业感源自分析质量，绝非来自排版。

## 色彩方案

| 用途 | 颜色 |
|------|------|
| 正文文字 | 黑色（#000000） |
| 章节标题 | 黑色 — 下划线（叙事型）或加粗（结构化型） |
| 分割线 | 黑色，0.5pt |
| 页脚文字 | 黑色 |
| 超链接 | 黑色（或蓝色 #0000EE，如为超链接） |

**不使用任何其他颜色。不使用灰色文字、彩色背景、强调色、阴影或填充。**

## 字体排版系统

### 字体家族

**Times New Roman**（或等效系统衬线字体：Georgia、Cambria）用于**所有文字**——正文、标题、引用块、页脚、图注、表格。文档内不混用不同字体家族。

叙事型备忘录参考使用的字体：
- TimesNewRomanPSMT（常规正文）
- TimesNewRomanPS-BoldMT（加粗强调）
- TimesNewRomanPS-ItalicMT（斜体引用、图注）

### 字号层级

| 元素 | 字号 | 样式 |
|------|------|------|
| 备忘录头部信息（Re: 行） | 12pt | 常规 |
| 章节标题 | 11-12pt | 下划线（叙事型）或加粗（结构化型） |
| 正文 | 11pt | 常规 |
| 正文内加粗强调 | 11pt | 加粗 |
| 引用块 / 缩进文字 | 11pt | 斜体 |
| 项目符号列表 | 11pt | 常规 |
| 页脚（版权、页码） | 9-10pt | 常规（页码可加粗） |
| 法律免责声明 | 9pt | 斜体 |
| 图表标注 | 10pt | 斜体 |

### 行间距

- 正文段落：1.15 至 1.5 倍行距
- 段后间距：6-12pt
- 标题：与正文连贯排列，标题前无额外间距，标题后 6-12pt
- 引用块：1.15 倍行距，左缩进约 0.5 英寸

## 页面布局

### 页边距

- 上：1 英寸（2.54 厘米）
- 下：1 英寸（2.54 厘米）
- 左：1 英寸（2.54 厘米）
- 右：1 英寸（2.54 厘米）

### 封面页

**无。** 文档直接以备忘录头部信息块开始。不设标题页、装饰封面或图形元素。

### 备忘录头部信息块（第 1 页）

```
Memo to:     [收件人]
From:        [作者]
Re:          [主题]
Date:        [日期]
________________________________________________________________________
```

标签（"Memo to:"、"From:"、"Re:"、"Date:"）左对齐。值紧跟标签后方，保持统一缩进（制表符对齐）。头部信息下方以横跨全宽的水平分割线分隔。

无"Memo to:"的变体：仅用于交易型备忘录的紧凑单行头部：
```
To: [收件人]    From: [作者]    Re: [主题]    Date: [日期]
________________________________________________________________________
```

### 正文布局

- 单栏全宽排版
- 左对齐（非两端对齐）
- 段首不缩进；使用段间距代替
- 文字连续流转——各章节间直接衔接，不插入分页

### 页眉

**无。** 页边距内不设章节名称页眉。页眉留空。

### 页脚

三段式页脚横跨底部：
```
© [年份] [机构名称]                              [第 N 页]          版权所有
```
- 左侧：版权声明（如"© [年份] [机构名称]"）
- 居中：页码加粗（如"2"）
- 右侧："版权所有"

涉及保密的交易型备忘录，使用：
```
机密 — 仅供内部使用                              [第 N 页]
```

## 章节标题样式

### 叙事型/投资人信函风格（默认）

章节标题为**下划线文字独占一行**，融入文档正文流中：

```
...上一段落末尾。
理解AI

在进入正题之前...
```

特征：
- 下划线，与正文字号相同或略大（11-12pt）
- 独占一行，后接空行，然后是正文
- 不编号
- 不加粗（除非加粗是唯一的强调方式；下划线是主要标记）
- 不使用大号字体或不同颜色
- 标题前不分页

叙事型章节标题示例：
- "理解AI"
- "AI能思考吗？"
- "AI领域的最新进展"
- "法律信息与披露"（末尾章节，斜体+下划线）

### 结构化/交易型备忘录风格（备选）

风投交易型备忘录使用**加粗文字标题**（顶级风投惯例）：

```
简介

[公司]是一个值得关注的种子轮投资机会...

交易条款

我们的方案是在种子轮投入100万美元...
```

特征：
- 加粗，11-12pt，独占一行
- 后接空行，然后是正文
- 不编号（不使用"1."、"2."、"3."前缀）
- 子标题可使用加粗且字号稍小（如"公司使命："、"核心问题："）

## 文字处理规则

### 加粗

在段落**内部**使用加粗进行强调，而非用于标题（除非采用结构化风格）：
- 关键结论或建议
- 锚定读者注意力的重要短语
- 指标亮点
- 开篇论点句

示例："**首先是AI发展的速度。** 这种速度前所未有……"

### 斜体

- 引用块和第三方摘录
- 图表标注
- 法律免责声明
- 公司/产品名称首次提及时（可选惯例）
- 法律/免责声明章节标题："*法律信息与披露*"并加下划线

### 下划线

- 叙事型备忘录中的章节标题（这是**主要**标题样式）
- 正文强调不使用下划线

### 引用/缩进段落

- 左缩进：距左边距 0.5 英寸
- 字体：斜体，11pt，行距与正文相同
- 块级引用不加引号
- 不加左边框或背景阴影
- 不加彩色装饰条
- 出处另起一行，前加破折号，右对齐或行内排列

示例：
```
        有人专门为你设计了九个模块的课程，围绕你十二月的备忘录、
        你的智识框架来构建……

        —[来源出处]
```
## 表格样式

两份参考备忘录均避免使用正式表格。叙事型备忘录**完全没有表格**。数据应以以下方式呈现：

- 行内文字配合加粗关键数值
- 项目符号列表配合缩进子项
- 简洁的缩进段落用于结构化数据

如确实需要使用表格（如交易型备忘录中的财务摘要）：
- 仅表头行和外边框使用细黑线
- 不使用颜色填充（表头行仅用加粗文字）
- 全部使用黑色文字
- 10pt 字号

## 图表样式

- 图表嵌入正文中，不单独分页
- 使用简单的折线图或基础图形；不使用花哨的图表样式
- 图表标签置于图表上方："图表 1：[描述]"——加粗或斜体，10pt
- 图表周围不加彩色背景或边框
- 正文中引用："（见图表 1）"或"如图表 1 所示"

## 保密与法律标记

当备忘录包含敏感信息时，仅在页脚添加标记：
- 左对齐："机密——仅供内部使用"或"高度机密"
- 9pt，大写或小型大写字母

文档末尾的法律免责声明：
- 斜体，9pt，左对齐
- 前以水平分割线分隔
- 标题："*法律信息与披露*"并加下划线
- 标准免责声明内容：观点可能变更、过往业绩不代表未来表现、非要约邀请、信息来源说明

## 中日韩字体排版

如备忘录包含中文/日文/韩文内容，将 Times New Roman 替换为支持 CJK 的衬线字体：Noto Serif CJK、思源宋体或宋体。该替换统一应用于所有文本元素。
