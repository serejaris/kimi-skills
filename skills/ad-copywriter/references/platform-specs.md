# 平台规格参考

各广告平台的完整字符限制、格式要求和最佳实践。

---

## Google Ads

### 自适应搜索广告（RSA）

| 元素 | 字符限制 | 是否必填 | 说明 |
|------|----------|----------|------|
| 标题 | 30字符 | 最少3条，最多15条 | 任意3条可能同时展示 |
| 描述 | 90字符 | 最少2条，最多4条 | 任意2条可能同时展示 |
| 显示路径1 | 15字符 | 可选 | 显示在域名后的URL中 |
| 显示路径2 | 15字符 | 可选 | 显示在路径1之后 |
| 最终URL | 无限制 | 必填 | 落地页URL |

**组合规则：**
- 系统会选择最多3条标题和2条描述进行展示
- 标题之间以" | "分隔或堆叠显示
- 除非固定位置，否则任何标题可出现在任何位置
- 固定位置会降低系统优化能力——谨慎使用

**固定策略：**
- 如品牌规范要求，将品牌名固定在位置1
- 将最强CTA固定在位置2或3
- 其余标题不固定，交给机器学习优化

**标题组合建议（15条标题）：**
- 3-4条关键词导向（匹配搜索意图）
- 3-4条利益点导向（用户能获得什么）
- 2-3条社会认同（数字、奖项、客户数）
- 2-3条CTA导向（引导行动）
- 1-2条差异化（为什么选你而非竞品）
- 1条品牌名标题

**描述组合建议（4条描述）：**
- 1条利益点+证据
- 1条功能+成果
- 1条社会认同+CTA
- 1条紧迫感/优惠+CTA（如适用）

### 效果最大化广告（Performance Max）

| 元素 | 字符限制 | 说明 |
|------|----------|------|
| 标题 | 30字符（需5条） | 适配各版位的短标题 |
| 长标题 | 90字符（需5条） | 用于展示广告、视频、发现 |
| 描述 | 90字符（必填1条，最多5条） | 配合各种广告格式 |
| 商家名称 | 25字符 | 必填 |

### 展示广告

| 元素 | 字符限制 |
|------|----------|
| 标题 | 30字符 |
| 长标题 | 90字符 |
| 描述 | 90字符 |
| 商家名称 | 25字符 |

## Meta 广告（Facebook & Instagram）

### 单图/视频/轮播

| 元素 | 建议长度 | 最大长度 | 说明 |
|------|----------|----------|------|
| 正文 | 125字符 | 2,200字符 | 图片上方文字；约125字符后被截断 |
| 标题 | 40字符 | 255字符 | 图片下方；约40字符后被截断 |
| 描述 | 30字符 | 255字符 | 标题下方；不一定展示 |
| 显示链接 | 40字符 | 无限制 | 可选的自定义显示URL |

**版位说明：**
- **信息流**：所有元素均展示；正文最显眼
- **快拍/Reels**：正文覆盖在画面上；控制在72字符以内
- **右侧栏**：仅标题可见；无需描述
- **Audience Network**：因发布商而异

**最佳实践：**
- 正文开头直接抛出钩子（前125字符）
- 较长正文使用换行提升可读性
- 表情符号：可测试但不要滥用——每条广告最多1-2个
- 正文中使用提问句可提升互动
- 标题应是清晰的CTA或价值陈述

### 线索广告（即时表单）

| 元素 | 限制 |
|------|------|
| 问候标题 | 60字符 |
| 问候描述 | 360字符 |
| 隐私政策文案 | 200字符 |

| 元素 | 限制 |
|------|------|

**LinkedIn 平台指南：**
- 语气专业但不枯燥

### Spark 广告（加热原生内容）

| 元素 | 说明 |
|------|------|
| 文案 | 使用原帖文案 |
| CTA按钮 | 由广告主添加 |
| 展示名称 | 原创作者账号 |

**TikTok 平台指南：**
- 原生风格内容效果优于精致广告
- 前2秒决定用户是否继续观看
- 使用热门音乐和格式
- 文字叠加必不可少（大部分用户静音观看）
- 仅竖版视频（9:16）

**Twitter/X 平台指南：**

- 短句效果最好

- 话题标签：最多1-2个（广告中0个往往更好）
- 长推文串适合决策阶段的内容

## 字符计数注意事项

- 所有平台上**空格**都算字符
- **表情符号**根据平台不同计为1-2个字符
- **特殊字符**（|、&等）计为1个字符
- 正文中的**URL**计入字符限制
- **动态关键词插入**（`{KeyWord:default}`）可能超出限制——设置安全的默认值
- 上线前务必在平台的广告预览中确认

## 多平台素材适配

同时为多个平台创作时，从限制最严格的格式开始：

1. **Google搜索标题**（30字符）——迫使你提炼最精简的信息
2. **扩展为Meta标题**（40字符）——增加一两个词
3. **扩展为LinkedIn引言**（150字符）——补充背景和证据
4. **扩展为Meta正文**（125+字符）——完整的钩子和价值主张

这种层层递进的方式确保核心信息在所有平台都适用，再根据各平台允许的空间进行丰富。
