# 主题工厂 Skill

本 skill 提供一套精心策划的专业字体与配色主题合集，每个主题都包含精选的色彩方案和字体搭配。选定主题后，即可将其应用到任意内容上。

## 用途

使用本 skill 为演示文稿幻灯片或其他内容应用一致且专业的视觉风格。每个主题包括：
- 一组协调统一的配色方案（含十六进制色值）
- 相互搭配的标题与正文字体组合
- 独特的视觉风格，适用于不同场景和受众

## 使用说明

为幻灯片或其他内容应用样式的步骤：

1. **展示主题预览**：向用户展示 `theme-showcase.pdf` 文件，让用户直观浏览所有可选主题。请勿对该文件做任何修改，仅供查看。
2. **询问用户选择**：询问用户希望应用哪个主题
3. **等待确认**：获取用户对所选主题的明确确认
4. **应用主题**：主题确认后，将所选主题的配色和字体应用到内容上

## 可选主题

以下 10 种主题均可在 `theme-showcase.pdf` 中预览：

1. **深海蓝韵（Ocean Depths）** — 专业沉稳的海洋风格
2. **落日大道（Sunset Boulevard）** — 温暖鲜明的日落色调
3. **森林华盖（Forest Canopy）** — 自然质朴的大地色系
4. **现代极简（Modern Minimalist）** — 简洁现代的灰阶配色
5. **黄金时刻（Golden Hour）** — 浓郁温暖的秋日风情
6. **极地冰霜（Arctic Frost）** — 清冷爽利的冬日主题
7. **沙漠玫瑰（Desert Rose）** — 柔和雅致的复古色调
8. **科技前沿（Tech Innovation）** — 大胆前卫的科技美学
9. **植物花园（Botanical Garden）** — 清新自然的花园配色
10. **午夜星河（Midnight Galaxy）** — 深邃神秘的宇宙色调

## 主题详情

每个主题在 `themes/` 目录下都有完整的规格定义，包括：
- 协调统一的配色方案（含十六进制色值）
- 相互搭配的标题与正文字体组合
- 独特的视觉风格，适用于不同场景和受众

## 应用流程

选定心仪的主题后：
1. 从 `themes/` 目录读取对应的主题文件
2. 将指定的配色和字体统一应用到整个内容中
3. 确保对比度和可读性达标
4. 在所有页面中保持主题视觉风格的一致性

## 创建自定义主题
如果现有主题都不符合需求，可以创建自定义主题。根据用户提供的需求描述，生成一个与上述主题风格一致的新主题。为主题起一个能描述其字体/配色特征的名称，根据用户的描述选择合适的配色与字体。生成后先展示给用户审阅确认，确认后再按上述流程应用主题。
