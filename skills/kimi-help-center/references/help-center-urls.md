# Kimi Help Center URL Index

## URL Construction

Select the base URL based on user account type, then append the relative path:

| Account Type | Base URL | Description |
|-------------|----------|-------------|
| Mainland China (+86 phone) | `https://www.kimi.com/zh-cn/help/` | Chinese help center |
| Global/Overseas (non-+86, Google, etc.) | `https://www.kimi.com/help/` | English help center |

**Formula**: `Base URL` + `relative path` = `full URL`

**Examples**:
- China user: `https://www.kimi.com/zh-cn/help/` + `getting-started/overview` → `https://www.kimi.com/zh-cn/help/getting-started/overview`
- Global user: `https://www.kimi.com/help/` + `getting-started/overview` → `https://www.kimi.com/help/getting-started/overview`
- With anchor: `https://www.kimi.com/zh-cn/help/` + `membership/update-rules#h-36gipa` → `https://www.kimi.com/zh-cn/help/membership/update-rules#h-36gipa`

**Note**: The same article may have different paths and anchor IDs in Chinese vs English. Use the corresponding column in `routing-tables.md`.

**Fallback**: When account type cannot be determined, default to `zh-cn` for Chinese questions and `help` for English.
