## Product Overview

Kimi is a multimodal AI chat product by Moonshot AI.

### Platform Coverage

| Platform | Description |
|----------|-------------|
| Mobile App | Native apps for Android, iOS, and HarmonyOS |
| Web | Browser access at kimi.com |
| Desktop Client | macOS and Windows |
| WeChat Mini Program | Embedded in the WeChat ecosystem |
| Browser Extension | Browser add-on |

### Input Methods

- Text input
- File parsing (PDF / Word / Excel / PPT / TXT / images / video)
- Voice input

### Core Feature Modules

| Module | Description | Help Center Category |
|--------|-------------|---------------------|
| Core model chat (K2.6) | Core conversation capability | Getting Started |
| Thinking mode (K2.6) | Deep reasoning | Getting Started |
| Agent (OK Computer) | Autonomous task execution | Agent Mode |

### User System

- Account security (phone / email / Google login)
- Free tier: Adagio
- **China paid tiers**: Andante / Moderato / Allegretto / Allegro
- **Global paid tiers**: Moderato / Allegretto / Allegro / Vivace
- Chat history
- Session limits

### ToB / Developer Services

- **Kimi API**: Enterprise-facing model API service (platform.kimi.com)
- **Kimi Code**: A coding-focused perk of the Kimi membership, engineered to drop into any dev workflow, answer faster and more reliably, and get programming tasks done in record time.
