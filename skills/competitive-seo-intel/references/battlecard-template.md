# 竞争情报战斗卡片模板

为销售、营销和内容团队提供可操作竞争情报的填空式模板。建议为每个主要竞品维护一张战斗卡片，至少每季度审核一次。

## 概述

竞争情报战斗卡片是一份简明的参考文档，汇总竞品的优势、劣势、定位以及如何与其竞争。战斗卡片是活文档，应随竞品变化及时更新。

**目标受众：** 销售团队、内容策略师、营销管理层
**更新频率：** 至少每季度审核一次；竞品有重大变化时立即更新
**篇幅目标：** 每个竞品 2-3 页（可快速浏览，非面面俱到）

### 第九部分：埋雷问题

在销售流程早期向潜在客户提出的问题，用于凸显你的优势并暴露竞品弱点。适合在需求挖掘电话中使用。

| 埋雷问题 | 能暴露什么 | 你的优势 |
|------------------|----------------|---------------|
| "[你有而对方没有的能力] 对你的工作流有多重要？" | 为你的差异点创造需求 | [你的能力] |
| "你是否遇到过 [竞品的常见痛点]？" | 暴露竞品已知弱点 | [你对该痛点的解决方案] |
| "你期望多久能看到这个工具的投资回报？" | 凸显见效时间差异 | [你更快的见效时间] |
| "你的团队目前如何处理 [你擅长的使用场景]？" | 引导到你擅长的领域 | [你在此场景的方案] |

## 战斗卡片维护指南

### 更新触发条件

以下情况发生时应立即更新战斗卡片：

| 触发事件 | 需要更新的内容 |
|---------|---------------|
| 竞品发布新功能 | 功能对比、差异点、埋雷问题 |
| 竞品调整定价 | 定价对比、异议处理 |
| 竞品获得融资/上市 | 概览部分、定位 |
| 评论趋势发生重大变化 | 劣势、异议处理、胜败分析 |
| 你发布对标功能 | 差异点、功能对比 |
| 与该竞品的重要胜败 | 胜败分析、异议处理 |
| 竞品发布重要内容 | SEO/内容情报 |

### 季度审核清单

- [ ] 确认所有定价信息是最新的
- [ ] 检查竞品是否发布了新功能
- [ ] 查看最新的 G2/Capterra/TrustRadius 评论是否出现新趋势
- [ ] 用近期成交更新胜败分析
- [ ] 刷新 SEO/内容指标
- [ ] 确认差异点仍然准确
- [ ] 与销售团队验证所有埋雷问题的相关性
- [ ] 归档过时信息（不要删除——移至"历史"部分）

### 战斗卡片更新数据来源

| 来源 | 收集什么 | 更新频率 |
|--------|---------------|-----------------|
| 竞品网站 | 定价、功能、信息传递、定位 | 每月 |
| G2/Capterra 评论 | 优势、劣势、客户情绪 | 每季度 |
| LinkedIn/社交媒体 | 团队变动、公司动态、内容策略 | 每月 |
| 行业媒体/博客 | 融资、合作、市场认知 | 发布时 |
| 销售团队反馈 | 胜败主题、异议模式 | 每月 |
| SEO 工具 | 流量、排名、外链、内容变化 | 每月 |
| AI 系统查询 | AI 引用模式、品牌提及 | 每季度 |
| 客户反馈 | 竞品切换原因 | 每季度 |
