# 竞品分析 — 分析模板

各竞品分析步骤的标准模板，用于规范输出格式。

```markdown
## 竞品档案：[名称]

**基本信息**
- 网址：[website]
- 域名年龄：[years]
- 预估流量：[monthly visits]
- 域名权威度/评级：[score]

**商业模式**
- 类型：[SaaS/电商/内容/等]
- 目标受众：[description]
- 核心产品/服务：[products/services]
```
