# 竞品分析 — 示例报告

这是一份完整的示例输出，展示竞品分析交付物的预期格式。

## 提示词

**用户**："分析一下 HubSpot 为什么在营销类关键词上排名这么好"

## 示例输出

```markdown
# 竞品分析：HubSpot

## SEO 策略概览

HubSpot 通过以下组合策略主导了营销类关键词：
1. **海量内容护城河** - 10,000+ 篇博客文章
2. **免费工具引流** - 网站评分器、邮件签名生成器
3. **教育品牌** - 学院、认证、在线课程
4. **主题集群模型** - 率先采用支柱页/集群的内容组织方式

## 成功的关键因素

### 内容策略

**发布量**：每月 50-100 篇
**平均字数**：2,500+ 字
**内容类型**：
- 深度指南（35%）
- 操作教程（25%）
- 模板和示例（20%）
- 数据/研究（10%）
- 工具和计算器（10%）

**高流量内容规律**：
1. 广泛主题的终极指南
2. 需邮箱订阅的免费模板
3. 数据统计合集文章
4. 术语解释文章（"什么是[某术语]"）

### GEO 成功因素

HubSpot 频繁出现在 AI 回答中，原因如下：

1. **每篇文章开头都有清晰的定义**
   > "集客营销（Inbound Marketing）是一种通过创造有价值的内容和个性化体验来吸引客户的商业方法论。"

2. **可引用的统计数据**
   > "坚持写博客的公司网站访问量增加 55%"

3. **全面的内容覆盖** - AI 信任其权威性

### 可链接资源

| 资源 | 外链数 | 为什么有效 |
|------|--------|------------|
| 网站评分器 | 45,000+ | 免费、即时获得价值 |
| 营销统计数据 | 12,000+ | 可引用的参考资料 |
| 博客创意生成器 | 8,500+ | 解决真实痛点 |

## 可利用的劣势

1. **内容日渐陈旧** - 许多文章超过 3 年未更新
2. **建议过于笼统** - 缺乏行业纵深
3. **偏重企业级客户** - 对个人创业者服务不足
4. **创新缓慢** - 多年沿用相同的内容形式

## 你的机会

1. 创建他们无法覆盖的细分垂直内容
2. 瞄准他们忽略的长尾关键词
3. 在新兴领域构建交互式工具
4. 提供他们没有的原创研究
5. 在你的细分领域重点优化适合 GEO 的定义式内容
```
