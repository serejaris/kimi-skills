# 商业计划书路演PPT技能

创建精美、面向投资人的商业计划书/融资方案演示文稿，遵循经过验证的专业模板风格。

## 参考来源

- **参考类型**：用户上传的文件（PDF）
- **参考文件类型**：PDF（18页中国电力电子公司创业融资路演PPT）
- **有效参考文件类型**：PPTX（幻灯片演示文稿）
- **主要语言**：简体中文，穿插英文技术术语
- **参考字体**：楷体（STKaiti）用于中文正文/标题；Arial用于英文/数字；微软雅黑粗体用于强调

## 支持的输出格式

| 格式 | 支持情况 | 说明 |
|------|----------|------|
| PPTX | 主要格式 | 默认输出；可编辑的幻灯片 |
| PDF | 支持 | 从PPTX导出，用于分发 |

## 默认输出格式选择

1. 如果用户明确指定了格式，使用该格式。
2. 否则，默认输出**PPTX**（可编辑的演示文稿）。
3. 如果用户要求"可分享的"/"最终版"，可同时提供PDF。

## 工作流程：创建路演PPT

### 步骤一 — 收集信息

从用户处收集以下内容：
- 公司名称、logo、标语
- 所属行业/赛道
- 融资轮次和目标金额
- 关键指标（营收、增长率、利润率、团队规模、专利数量）
- 产品/服务描述 + 图片（如有）
- 竞争格局（3-5家同行）
- 财务预测（3-5年）
- 团队简介 + 照片
- 市场规模/TAM数据

如果用户提供了文字简介或已有文档，从中提取上述信息。

### 步骤二 — 规划幻灯片结构

将用户内容映射到标准章节流程。阅读 `references/structure_contract.md` 获取完整的层级结构和布局模式。默认的14个章节流程如下：

1. **封面** — 公司标识 + 投资机构品牌 + 视觉冲击力图片
2. **投资亮点** — 6宫格展示核心卖点
3. **市场机遇** — 市场规模图表 + 增长逻辑
4. **行业格局** — 竞争对手图谱/同行对比
5. **公司定位** — 战略支柱/价值主张
6. **核心竞争力** — 优势矩阵
7. **产品组合** — 产品概览 + 技术架构
8. **技术差异化** — 技术路线对比 + 参数表格
9. **团队介绍** — 创始人 + 核心团队 + 顾问委员会
10. **增长预测** — 营收/订单预测图表
11. **愿景/路线图** — 多阶段战略规划
12. **融资方案** — 融资需求 + 资金用途分配
13. **附录** — 支撑数据、产品图片、详细对比
14. **免责声明** — 法律声明 + 联系方式

**适配规则**：跳过用户没有相关内容的章节。如果幻灯片总数超过20页，可将紧密相关的章节合并。对于种子轮公司，第7-9章可以精简；对于成长期公司，第10-12章应更详尽。

**关键规则**：参考模板采用平铺式的18页结构，各主要章节之间**没有过渡页**。请勿添加章节分隔页。

### 步骤三 — 应用视觉风格

阅读 `references/style_contract.md` 获取完整的风格规范。核心规则：

- **背景**：所有内容页均为白色（`#FFFFFF`）。绝不使用深色背景。
- **配色方案**：主色调为藏青色 `#1B3A6B`（深蓝色）。绿色**仅**用于图表的次要数据系列（如海外/辅助数据的绿色柱状图）。绿色不作为主色调。
- **页眉**：每页内容页标题下方有一条藏青色水平线（2px）。公司logo固定在每页内容页的右上角。
- **数据可视化**：柱状图/条形图的**主色**为藏青色。绿色仅用于次要数据系列。环形图使用藏青色+绿色（或藏青色+灰色）。
- **强调样式**：关键指标加粗，字号比周围文字大2-4pt。
- **总结横幅**：页面底部全宽藏青色矩形，白色文字展示核心结论。

### 步骤四 — 设计每页幻灯片

为每页幻灯片选择 `references/structure_contract.md` 中定义的布局模式：

- **宫格亮点**（模式A）— 3×2卡片，包含图标、标题和两行要点摘要
- **双图表**（模式B）— 左右并列的两个图表，带深色标题栏
- **对比流程**（模式C）— 水平箭头流程 + 下方对比表格
- **战略支柱**（模式D）— 纵向堆叠：藏青色标签胶囊 + 每行要点列表
- **三栏矩阵**（模式E）— 三个并列模块 + 底部图标网格
- **上下对比**（模式F）— 上下流程图 + 标注框
- **图片网格**（模式G）— 三栏产品图片 + 说明文字
- **底部总结横幅**（模式H）— 藏青色全宽总结横幅

每页设计原则：
- 一页一个核心信息
- 正文最多40-50个汉字或约80个英文单词
- 3-6个高亮数据点
- 优先使用图表/图示，而非文字要点
- 每个论点都应有数据、图表或来源支撑

### 步骤五 — 审查与润色

- 确保所有内容页均为白色背景——无例外
- 确认藏青色在所有页面中均为主色调
- 检查所有图表均以藏青色为主要填充色
- 确认logo位置（右上角）在每页内容页中保持一致
- 确认每页标题下方有藏青色下划线
- 验证文字不会溢出形状边界
- 确认团队成员照片为真实照片，而非通用图标
- 确认产品图片为真实图片，而非占位符
- 请勿添加"严格保密"水印或页脚

## 幻灯片类型速查表

| 幻灯片类型 | 推荐布局模式 | 关键视觉元素 |
|---|---|---|
| 封面 | 1/3白色 + 2/3图片的分割布局 | Logo、标题、副标题、投资机构logo、右侧主图 |
| 亮点 | 模式A（宫格） | 6个图标卡片、关键数据 |
| 市场规模 | 模式B（双图表） | 藏青色柱状图、数据来源标注 |
| 竞争对手 | 模式C（对比流程） | 箭头流程、同行logo、对比表格 |
| 公司定位 | 模式D（战略支柱） | 藏青色标签胶囊 + 要点列表 |
| 核心优势 | 模式E（三栏矩阵） | 并列模块、图标网格 |
| 技术差异化 | 模式F（上下对比） | 流程图、标注框 |
| 团队 | 自定义：照片 + 简历卡片 | 真实头像照片、姓名、职务、简历要点 |
| 财务预测 | 模式B（双图表） | 藏青色柱状图 + 藏青/绿色饼图 |
| 路线图 | 分阶段展示 | 圆角矩形、箭头、阶段编号 |
| 融资需求 | 自定义：居中大数字 + 4个环形图 | 大额金额、4个独立环形图、用途标签 |
| 产品图片 | 模式G（图片网格） | 3张真实产品图片 + 说明文字 |
| 对比表格 | 自定义：全宽表格 | 藏青色表头、斑马纹行、首列加粗 |
| 免责声明 | 单栏文字为主 | 大标题、要点式声明、联系方式网格 |

## 中文内容字体策略

当演示文稿包含中文文字时（默认情况）：
- **标题字体**：楷体 / STKaiti（替代方案为思源宋体SC）
- **正文字体**：与标题字体一致（保持单一字体族的统一性）
- **英文/数字补充**：Arial（用于英文术语和数字）
- **加粗强调**：使用该字体的粗体字重，或切换到支持粗体的中文字体（如微软雅黑粗体）

当演示文稿为纯英文时：
- **标题字体**：Georgia 或 Merriweather（衬线体，专业感）
- **正文字体**：与标题同一字体族
- **强调/加粗**：使用同一字体族的粗体字重
