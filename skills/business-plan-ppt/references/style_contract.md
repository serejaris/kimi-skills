# 风格规范：商业计划书路演PPT

本文档定义了从参考模板（一份18页的中国电力电子创业公司融资计划书/BP PDF）中提取的可复用视觉风格规则。该规范与语言无关——制作英文版PPT时选择对应的等效字体即可。

## 关键风格规则（必须遵守）

1. **所有内容页均为白色背景** — 无例外。参考模板全程使用干净的白色背景。
2. **藏青色为主色调** — 不是绿色。绿色**仅**用于图表的次要数据系列。
3. **无章节分隔页** — 参考模板为平铺式18页结构，无章节过渡页。
4. **封面为分割式布局** — 左侧1/3白色面板 + 右侧2/3图片。
5. **融资页使用4个独立环形图** — 而非单一的甜甜圈图。
6. **团队页展示真实照片** — 不使用通用图标或占位符。
7. **产品页展示真实图片** — 不使用空白占位符。
8. **不添加"严格保密"水印** — 参考模板无此类水印。

## 参考字体与语言

- 主要文字系统：简体中文，穿插英文技术术语。
- 通过PDF字体提取识别的参考字体族：
  - **中文正文与标题**：楷体（STKaiti）— 一种衬线书法风格的中文字体，用于标题、副标题和正文。
  - **英文/数字补充**：ArialMT / Arial-BoldMT — 一种无衬线字体，用于数字、英文标注和公式。
  - **中文加粗强调**：MicrosoftYaHei-Bold（微软雅黑粗体）— 用于需要加粗强调的中文短语。
- 当内容不确定是否为纯英文时，建议整个演示文稿使用**一个统一的中文字体族**。楷体不可用时的替代方案：
  - 思源宋体（Noto Serif CJK SC）— 保留衬线书法风格。
  - 宋体（SimSun）— 广泛可用的中文衬线备选字体。
  - 微软雅黑（Microsoft YaHei）— 如需现代感可选择此无衬线中文字体。
- 纯英文演示文稿可将楷体替换为Georgia或Merriweather等简洁的衬线体。

## 配色方案

| 标识 | 色值 | 用途 |
|------|------|------|
| 主色藏青 | `#1B3A6B` | 标题栏、对比色块、关键数据、图标、图表主色填充、页标题下划线。这是**主导色**。 |
| 辅助蓝 | `#2C5885` | 标题栏背景、标签标题、高亮文字 |
| 辅助绿 | `#4A9B7F` | **仅用于**图表次要数据系列（如海外/辅助数据系列的绿色柱状图）。**不作为主色调**。 |
| 浅灰 | `#F5F5F5` | 特殊对比区域的背景填充 |
| 深色文字 | `#1A1A1A` | 正文文字、常规中文字符 |
| 中灰 | `#666666` | 副文字、注释、图表标签 |
| 浅灰背景 | `#F8F8F8` | 图表背景、表格背景 |
| 白色 | `#FFFFFF` | 页面主背景、深色元素上的文字 |
| 对比色块背景 | `#1B3A6B` | 模块标题/总结横幅的圆角矩形（内部白色文字） |
| 总结横幅 | `#1B3A6B` | 页面底部横幅，用于展示核心结论（内部白色文字） |

## 页面布局与间距

### 幻灯片比例
- 16:9宽屏（标准路演PPT比例）。

### 母版页模板（关键规则）
- **背景**：每页内容幻灯片均为白色（`#FFFFFF`）。任何地方都不使用深色背景。
- **顶部区域**：页标题位于左上角。样式：22-28pt，加粗，深色文字色。
- **标题下划线**：一条藏青色水平线（2px，`#1B3A6B`），在标题区域下方横贯整个幻灯片宽度。
- **右上角logo**：公司logo，约120 × 40 px，紧贴右边距顶部放置。**每页**内容幻灯片都必须出现。
- **安全边距**：左右约40px，上下约35px。
- **内容区域**：标题线下方全宽区域，底部留出空间放置可选的总结横幅。

### 内容密度
- 低至中等密度。每页幻灯片传达一个核心信息。
- 模块之间、文字块之间留有充足的留白。
- 每页不超过6个要点或3个图表面板。

## 视觉元素

### 圆角对比色块
- 形状：圆角矩形，圆角半径8-12px。
- 填充：主色藏青（`#1B3A6B`）。
- 内部文字：白色、居中、加粗、16-18pt。
- 用途：模块标签、页内章节标题、分类标签、核心结论横幅。

### 数据高亮文字
- 行内关键指标（如 `>50%`、`超10年`、`100%+`）：加粗并放大（比周围文字大2-4pt）。
- 频率：每页内容幻灯片2-4个高亮数字为最佳。

### 章节分隔线
- 一条细水平线（1-2px，主色藏青），用于在单页内分隔不同内容区块。

## 图表与数据可视化风格

### 柱状图/条形图
- **主数据系列填充**：藏青色（`#1B3A6B`）。这是所有柱状图的默认颜色。
- **次要数据系列填充**：辅助绿（`#4A9B7F`）— **仅**用于分组或堆叠图表中的次要/海外数据系列。
- 背景：白色或浅灰背景（`#F8F8F8`）。
- 标签：图表下方以10-11pt中灰色标注数据来源+机构名称。
- 分类轴标签：12-13pt深色文字。
- 数值轴：无轴线，网格线可选（浅灰虚线）。

### 环形图
- 内径：约为外径的55%。
- **融资页**：四个**独立**环形图（非单一甜甜圈图）。每个环形图显示一个资金分配比例（如30%研发、25%生产、25%产能扩充、20%市场拓展）。
- 重点扇区：主色藏青（`#1B3A6B`）。
- 辅助扇区：浅灰（`#E0E0E0`）。
- 中心标签：深色文字加粗显示百分比。
- 图例：位于下方或右侧，颜色与环形图匹配。

### 饼图
- **营收分布饼图**（70/30风格）：主色藏青（`#1B3A6B`）用于70%扇区，辅助绿（`#4A9B7F`）用于30%扇区。
- 扇区之间用白色分隔线（1px）。
- 百分比标签放在扇区内部（白色）或外部。

### 流程图/路线图
- 用箭头连接各阶段。
- 每个阶段容器：圆角矩形，带微妙的背景渐变或浅灰填充，白色边框。
- 阶段编号标记：顶部的小型主色藏青圆形。
- 内容：标题加粗，下方为常规字重的要点。
- 箭头连接线：主色藏青，尖三角箭头。

## 图标风格
- 风格：极简线条图标，单色（主色藏青或深色文字色）。
- 尺寸：用作圆形图标卡片时直径32-48px。
- 圆形图标容器：细圆形轮廓线，内含图标，放置在网格布局中功能标题的上方。

## 表格风格
- 表头行：主色藏青背景（`#1B3A6B`），白色加粗文字，适当内边距。
- 数据行：白色/`#F8F8F8` 交替（可选斑马纹）。
- 单元格边框：仅水平边框，1px实线`#E0E0E0`。
- 首列：当包含分类标签时加粗。
- 对齐：数字/指标居中，文字左对齐。垂直对齐：居中。
- 行高：最小36px，以适应中文文字。

## 页眉/页脚规则
- 内容页不显示页码。
- 右上角logo作为每页持续的品牌锚点。
- **不添加"严格保密"水印或页脚文字。**
- 最后1-2页可选页脚：保密声明 + 公司品牌 + 联系方式——但仅作为普通可见文字，不作为水印。

## 封面与末页

### 封面页（关键规则——分割式布局）
- 布局：分割式——左侧约1/3（白色背景）+ 右侧约2/3（满幅视觉冲击力图片）。
- 左侧面板：
  - 公司logo顶部对齐。
  - 主标题：超大号（36-48pt），加粗，两行，深色文字。
  - 副标题：20-24pt，中灰色。
  - 底部区域：投资机构/顾问公司logo。
- 右侧面板：高质量的产品或行业图片。图片靠近分割线的左侧部分应有干净的渐变融合边缘（或构图断点），确保左侧面板的文字可读。
- **封面不使用满幅深色图片背景。**

### 末页/免责声明页
- 白色背景，简洁布局。
- 主标题："声明"或同等表述，大号加粗。
- 内容：结构化的免责声明要点。
- 底部区域：两栏联系方式，投资机构logo。
