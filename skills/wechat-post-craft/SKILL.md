# WeChat Article Writer — 微信公众号文章写作知识库

帮助用户撰写高打开率、高转发率的微信公众号文章。覆盖标题党技巧、排版规范、引流CTA设计，适配不同公众号调性，从选题到发布全流程指导。

## Quick Start

用户只需提供：
1. **文章主题**：想写什么内容
2. **公众号调性**（可选）：专业严肃 / 情感温暖 / 干货实用 / 轻松娱乐
3. **目标读者**（可选）：职场人 / 宝妈 / 大学生 / 创业者 / 泛人群
4. **文章目的**（可选）：涨粉 / 带货 / 品牌传播 / 用户教育 / 纯内容
5. **字数要求**（可选）：短文（800字内）/ 中篇（1500-2500字）/ 长文（3000字+）

示例：
> "帮我写一篇公众号文章，主题是'为什么你的时间管理总是失败'，干货实用调性，面向职场人，目的是涨粉，中篇长度。"

## 一、公众号文章类型分类（Article Type Taxonomy）

### 1.1 六大文章类型

| 类型 | 核心诉求 | 典型选题 | 关键指标 |
|------|---------|---------|---------|
| 干货教程型 | 提供方法论和可操作步骤 | "3步搞定XX""XX完全指南" | 收藏率 |
| 观点输出型 | 表达态度、引发讨论 | "我为什么反对XX""XX的真相" | 在看/转发率 |
| 情感共鸣型 | 击中情绪、引发认同 | "那些XX的瞬间""致XX的你" | 转发率 |
| 热点解读型 | 借势热点、输出见解 | "XX事件背后的XX""从XX看XX" | 打开率 |
| 故事叙事型 | 讲好一个故事、传递价值 | "我花了XX年才明白的XX" | 完读率 |
| 盘点清单型 | 精选推荐、提供信息密度 | "XX必备的N个XX""2026年XX推荐" | 收藏率 |

### 1.2 文章类型判定

```
用户需求分析
  │
  ├─ 需要教会读者做某事        → 干货教程型
  ├─ 需要表达立场、讨论争议    → 观点输出型
  ├─ 需要打动人心、引发共鸣    → 情感共鸣型
  ├─ 需要蹭热点、追时效        → 热点解读型
  ├─ 需要讲述人物/品牌故事     → 故事叙事型
  ├─ 需要做合集/推荐/榜单      → 盘点清单型
  │
  └─ 多类型交叉 → 确定主类型 + 辅类型，主结构取主类型
```

## 二、标题党技巧（Title Craft）

标题决定打开率。公众号文章标题的核心目标：在信息流中被手指"刹车"点进来。

### 2.1 十大高点击率标题公式

| 编号 | 公式 | 原理 | 示例 |
|------|------|------|------|
| T1 | 数字+结果 | 具体化承诺 | "坚持这5个习惯，我一年瘦了30斤" |
| T2 | 颠覆认知 | 打破常识制造好奇 | "你以为的努力，其实是在浪费时间" |
| T3 | 悬念留白 | 信息缺口驱动点击 | "同事偷偷做了这件事，月底工资比我多了8000" |
| T4 | 痛点直击 | 精准命中焦虑 | "月薪5千和月薪5万的人，差距只在这一点" |
| T5 | 身份代入 | 让目标读者对号入座 | "30岁还没存到50万的人，看完这篇该慌了" |
| T6 | 热点嫁接 | 借势流量话题 | "从[热点事件]看普通人最该学的一个道理" |
| T7 | 权威背书 | 借用高势能人物/机构 | "谷歌内部都在用的时间管理法，今天免费教你" |
| T8 | 对比冲突 | 制造戏剧张力 | "985毕业送外卖，高中辍学年入百万" |
| T9 | 清单诱惑 | 暗示信息密度高 | "2026年最值得考的10个证书（附备考攻略）" |
| T10 | 情绪共鸣 | 说出读者心里话 | "成年人最大的崩溃，是从缺钱开始的" |

### 2.2 标题优化四原则

```
原则一：前13个字定生死
  → 手机信息流只展示约13个中文字符
  → 核心卖点必须在前13个字出现
  → 反例："在这个充满挑战的时代里我想说一些关于职场的真心话" ✗
  → 正例："职场10年，这3条经验我后悔没早知道" ✓

原则二：具体胜过模糊
  → "很多钱" → "月入3万"
  → "很快" → "7天"
  → "一些方法" → "5个方法"

原则三：二选一测试法
  → 写完标题后，再想一个备选
  → 发给朋友问"你会点哪个"
  → 选被点击的那个

原则四：避免自嗨
  → 你觉得有趣 ≠ 读者觉得有趣
  → 标题的服务对象是读者，不是作者
  → 时刻反问：如果我在刷手机，看到这个标题会停下来吗？
```

### 2.3 各文章类型的推荐标题公式

| 文章类型 | 首选公式 | 次选公式 | 标题示例 |
|---------|---------|---------|---------|
| 干货教程型 | T1 数字+结果 | T9 清单诱惑 | "学会这3个Excel函数，你能早下班2小时" |
| 观点输出型 | T2 颠覆认知 | T4 痛点直击 | "别再鼓吹副业了，大多数人连主业都没做好" |
| 情感共鸣型 | T10 情绪共鸣 | T5 身份代入 | "被裁那天我没敢告诉家人，自己在车里坐了两小时" |
| 热点解读型 | T6 热点嫁接 | T2 颠覆认知 | "全网都在骂[某事件]，但没人说到最关键的一点" |
| 故事叙事型 | T3 悬念留白 | T8 对比冲突 | "她从月薪3千到年入千万，只因做对了一个选择" |
| 盘点清单型 | T9 清单诱惑 | T1 数字+结果 | "私藏3年的15个免费神器，今天全部分享出来" |

### 2.4 标题禁忌

| 禁忌 | 原因 | 示例 |
|------|------|------|
| 标题过长（超25字） | 信息流截断，核心信息丢失 | ✗ "在互联网行业工作了十年之后我总结出了一些关于如何提升自己核心竞争力的方法论" |
| 纯疑问句无利益点 | 读者没动力去找答案 | ✗ "你知道怎么学英语吗？" |
| 多个标点堆叠 | 显得不专业、像垃圾信息 | ✗ "震惊！！！99%的人都不知道！！！" |
| 内容与标题不符 | 信任透支，长期掉粉 | 标题说"月入10万"，正文全是鸡汤 |
| 使用生僻词/专业缩写 | 目标读者看不懂 | ✗ "浅析OODA循环在GTD框架下的synergy" |

## 三、公众号排版规范（Formatting Standards）

公众号排版直接影响完读率。核心原则：**让读者在手机屏幕上轻松阅读**。

### 3.1 基础排版参数

```
■ 正文字号：15-16px（推荐15px，手机阅读最舒适）
■ 字间距：1px 或 1.5px
■ 行间距：1.75 倍 或 2 倍（推荐1.75倍，疏密适中）
■ 段间距：段后留一行空行（视觉分段）
■ 页边距：左右各留 1 字宽度边距
■ 首行缩进：不缩进（公众号主流风格，缩进在手机上显得老派）
■ 正文颜色：#3f3f3f 或 #595959（不用纯黑#000000，太刺眼）
■ 强调颜色：与公众号主色调一致（通常用于加粗文字或重点标注）
```

### 3.2 排版元素使用指南

#### 标题层级

```
封面标题（推送标题）：最核心，决定打开率
  ↓
文内大标题（一级标题）：18px 加粗，用于大段落分隔
  ↓
文内小标题（二级标题）：16px 加粗，用于子话题分隔
  ↓
正文：15px 常规
  ↓
注释/来源：13-14px 灰色（#888888）
```

#### 分割线

```
使用场景：
  → 大段落之间的话题转换
  → 正文与引导语/CTA 之间的分隔
  → 头图下方与正文之间

样式选择：
  → 简约风格：一条细线 或 三个居中圆点 ···
  → 品牌风格：带公众号LOGO的定制分割线
  → 避免过于花哨的装饰性分割线（影响专业感）
```

#### 引用块

```
使用场景：
  → 金句/名人名言
  → 读者留言/用户反馈
  → 需要特别突出的核心观点

样式：左侧竖线 + 浅灰底色 + 斜体或正常字体
```

#### 图片使用

```
■ 封面图：比例 2.35:1（公众号规定），关键信息居中
■ 文内配图：宽度撑满，高度不超过手机一屏
■ 图片间距：图片上下各留一行空行
■ 图注：居中，13px，灰色
■ 图片数量：1500字文章搭配3-5张图，避免纯文字墙
■ GIF动图：控制在3MB以内，避免加载过慢
```

### 3.3 段落节奏（公众号专属）

公众号文章在手机上阅读，段落必须比传统文章更短：

```
黄金节奏：

[1句话独立成段] ← 金句/核心观点，视觉突出

[2-3句短段] ← 过渡或情绪铺垫

[4-6句中段] ← 展开论述，承载主要信息

[1句话独立成段] ← 转折/强调

[3-4句段落 + 加粗关键句] ← 干货部分

[配图 + 图注]

[2-3句收束段] ← 小结或过渡到下一部分
```

**核心规则：**
- 每段不超过手机上 4-5 行显示（约 80-120 字）
- 重要观点单独成段
- 加粗不超过全文的 5%（过多加粗 = 没有加粗）
- 每屏至少有一个"视觉锚点"（加粗、图片、引用块、小标题）

### 3.4 不同调性的排版差异

| 排版元素 | 专业严肃 | 情感温暖 | 干货实用 | 轻松娱乐 |
|---------|---------|---------|---------|---------|
| 字号 | 15px | 15px | 15px | 15-16px |
| 正文色 | #3f3f3f | #595959 | #3f3f3f | #595959 |
| 强调方式 | 加粗 | 加粗+强调色 | 加粗+高亮底色 | 加粗+emoji少量 |
| 段落长度 | 中等偏长 | 短段为主 | 中等+列表 | 极短段 |
| 图片风格 | 数据图表/信息图 | 暖色调/人物图 | 截图/教程步骤图 | 表情包/趣图 |
| 分割线 | 简约线条 | 装饰性分割线 | 简约线条 | 可爱/趣味分割线 |
| emoji使用 | 基本不用 | 少量点缀 | 作为列表标记 | 较多使用 |

## 四、引流CTA设计（Traffic & Conversion CTA）

CTA（Call To Action）是公众号文章的商业闭环。好的CTA让读者自然地完成你期望的动作。

### 4.1 公众号核心CTA类型

| CTA类型 | 目标动作 | 放置位置 | 触发心理 |
|---------|---------|---------|---------|
| 关注引导 | 点击关注公众号 | 文首/文末 | 持续获取价值的期待 |
| 转发引导 | 分享到朋友圈/群聊 | 文末 | 社交货币/利他心理 |
| 在看引导 | 点击"在看" | 文末 | 认同感/低成本支持 |
| 留言引导 | 写评论互动 | 文末 | 表达欲/被看见的需求 |
| 私域引导 | 添加微信/进社群 | 文末/文中 | 获取更多资源/归属感 |
| 产品引导 | 点击链接/小程序 | 文中/文末 | 解决痛点/占便宜心理 |
| 历史文章 | 点击往期内容 | 文末 | 深度了解/延伸阅读 |

### 4.2 CTA文案公式

#### 关注引导CTA

```
公式：价值承诺 + 更新频率 + 行动指令

示例：
→ "关注我，每周分享3个可落地的职场提升技巧。"
→ "点击上方蓝字关注，后台回复'资料'领取完整版思维导图。"
→ "第一次来的朋友，先点个关注，这里专写[你的领域]，不贩卖焦虑，只提供方法。"

进阶技巧：
→ 关注引导放在第一个有价值的知识点之后（读者刚获得价值，意愿最高）
→ 不要在文章开头第一句就要求关注（还没提供价值就索取）
```

#### 转发引导CTA

```
公式：情感驱动 或 利他驱动

示例：
→ "如果觉得有用，转发给你身边也在[痛点]的朋友吧。"
→ "看完点个转发，可能你的朋友正需要这篇文章。"
→ "这篇值得收藏和转发，下次遇到[场景]的时候翻出来看。"

核心原则：
→ 转发动机 = "分享这篇能让我在朋友眼中显得[聪明/有品位/善良/有用]"
→ 不要写"帮我转发一下"——读者没义务帮你
→ 要让转发成为读者自我表达的方式
```

#### 留言引导CTA

```
公式：低门槛问题 + 好奇心触发

示例：
→ "你在[主题]上踩过最大的坑是什么？留言区聊聊。"
→ "文中提到的5个方法，你觉得哪个最实用？"
→ "如果只能选一条建议给5年前的自己，你会说什么？"

技巧：
→ 问题要具体、易回答，不要问"你怎么看"这种大而空的问题
→ 可以在留言区提前精选几条优质评论，引导讨论方向
→ 承诺"精选留言我会逐一回复"能显著提高互动率
```

#### 私域引导CTA

```
公式：钩子资源 + 获取方式 + 稀缺感

示例：
→ "我整理了一份[资源名称]，添加我的微信（文末二维码），发送暗号'[关键词]'免费领取。"
→ "想加入我们的[主题]交流群？扫码添加小助手，备注'入群'。名额有限，满200人关闭。"
→ "后台回复'[关键词]'，领取本文提到的[资料/模板/工具]。"

注意事项：
→ 提供的资源必须与文章内容强相关（否则转化率低）
→ 稀缺感要真实，不要每篇都说"仅限今天"
→ 获取路径要简短：最多2步（回复关键词/扫码添加）
```

### 4.3 CTA放置策略

```
文章结构中的CTA布局：

[文章开头]
  → 可放轻量级关注引导（不超过1句）

[第一个知识点/故事高潮之后]
  → 最佳关注引导位置（读者刚获得价值）

[文章中段，干货最密集处]
  → 可嵌入私域引导（"完整版资料在社群"）
  → 可嵌入产品引导（自然融入解决方案）

[文章结尾，正文结束后]
  → 主CTA位置（转发/在看/留言三选一重点引导）
  → 辅CTA（历史文章推荐/关注引导）
  → 二维码/联系方式

重要原则：
  → 全文CTA不超过3处（过多引导 = 推销感 = 掉粉）
  → 主CTA只能有1个（同时要求转发+在看+留言 = 读者什么都不做）
  → CTA要与内容自然衔接，不要突兀地"硬切广告"
```

### 4.4 不同文章目标的CTA策略

| 文章目的 | 主CTA | 辅CTA | CTA调性 |
|---------|-------|-------|---------|
| 涨粉 | 关注引导 | 转发引导 | 强调持续价值 |
| 带货 | 产品引导 | 留言引导 | 解决痛点+限时优惠 |
| 品牌传播 | 转发引导 | 在看引导 | 强调社交货币 |
| 用户教育 | 收藏引导 | 私域引导 | 强调工具/资源价值 |
| 纯内容 | 在看/留言 | 历史文章 | 轻引导，重互动 |

## 五、公众号调性适配指南（Tone Adaptation）

### 5.1 四种主流调性及其特征

#### 专业严肃型

```
代表号：刘润、远川研究所、虎嗅
特征：
  → 用词精准，不用网络流行语
  → 数据支撑多，引用来源明确
  → 段落适中偏长，逻辑链完整
  → 观点鲜明但论证充分
  → 不用emoji，极少用感叹号
  → 配图以数据可视化、信息图为主

开头风格：
  "最近有一组数据值得关注：[数据]。这背后揭示的，是一个正在发生的结构性变化。"

结尾风格：
  "回到开头的问题，答案或许不是非此即彼，而是[核心洞察]。这才是我们真正需要思考的。"
```

#### 情感温暖型

```
代表号：十点读书、有书、夜听
特征：
  → 短句为主，节奏舒缓
  → 大量使用场景描写和细节
  → 故事驱动，先感动再讲道理
  → 第二人称"你"使用频繁，对话感强
  → 适度使用emoji点缀氛围
  → 配图暖色调，人物/生活场景

开头风格：
  "昨天深夜，收到一个读者的留言。她说：'我好像很久没有认真笑过了。'"

结尾风格：
  "愿你在疲惫的时候，记得有人在这里陪你。晚安，明天会更好。"
```

#### 干货实用型

```
代表号：秋叶PPT、半佛仙人、运营研究社
特征：
  → 结构清晰，多用小标题和列表
  → 步骤化、可操作
  → 加粗多，方便跳读和收藏
  → 案例+方法论交替
  → emoji作为列表标记（✅📌⚠️）
  → 配图以截图、流程图、对比图为主

开头风格：
  "今天分享一个我用了3年的[方法/工具/技巧]，简单但真的有效。"

结尾风格：
  "以上就是完整的操作流程。建议先收藏，用到的时候翻出来照着做。有问题留言区见。"
```

#### 轻松娱乐型

```
代表号：GQ实验室、新世相、网易哒哒
特征：
  → 段落极短，1-2句一段
  → 大量表情包和趣味配图
  → 网络流行语和段子
  → 排版活泼，花样多
  → 互动感强，像和朋友聊天
  → 脑洞大开，角度刁钻

开头风格：
  "我发现了一个残酷的真相：成年人之间的友谊，是从不回消息开始冷却的。"

结尾风格：
  "看到这里的都是真爱。点个在看，我继续给你们整活儿。"
```

### 5.2 调性一致性检查清单

写完文章后，检查是否保持了调性一致：

```
□ 标题风格与正文调性是否匹配？（严肃号不用"震惊体"）
□ 开头语气是否符合人设？
□ 用词风格全文一致？（不要前半段严肃后半段突然卖萌）
□ 配图风格与文字氛围匹配？
□ CTA语气与全文一致？（干货号不用"求求转发"）
□ emoji使用量符合调性？（专业号不堆emoji）
□ 段落长度符合调性特征？（情感号不写大段论证）
```

## 六、文章结构模板（Article Architecture）

### 6.1 通用文章结构

```
┌─────────────────────────────────────────┐
│  封面图 + 推送标题                        │
├─────────────────────────────────────────┤
│  开头钩子（前3句）                        │
│  → 决定读者是否继续往下看                 │
├─────────────────────────────────────────┤
│  核心内容（文章主体 70%）                 │
│  → 根据文章类型选择结构模板               │
├─────────────────────────────────────────┤
│  价值收束（结尾段）                       │
│  → 总结/升华/情感落点                     │
├─────────────────────────────────────────┤
│  CTA 引导区                              │
│  → 主CTA + 辅CTA                        │
├─────────────────────────────────────────┤
│  公众号信息区                             │
│  → 往期推荐 / 二维码 / 关注引导           │
└─────────────────────────────────────────┘
```

### 6.2 各类型文章结构模板

#### 模板 A：干货教程型

```
【标题】数字+结果 或 清单诱惑

【开头】痛点场景 + 承诺（"看完这篇你就会了"）

【正文】
  ■ 为什么这个问题重要（1-2段，建立动机）
  ■ 方法/步骤一（小标题 + 操作步骤 + 截图/示例）
  ■ 方法/步骤二
  ■ 方法/步骤三
  ■ 常见误区/避坑指南

【结尾】一句话总结 + 收藏引导

【CTA】"后台回复'XX'领取完整资料"
```

#### 模板 B：观点输出型

```
【标题】颠覆认知 或 痛点直击

【开头】抛出争议观点/现象

【正文】
  ■ 主流认知是什么（先描述"大家以为的"）
  ■ 为什么这个认知有问题（转折，制造冲突）
  ■ 我的观点 + 论据一（案例/数据）
  ■ 论据二（类比/逻辑推演）
  ■ 可能的质疑与回应（体现思考深度）

【结尾】把观点浓缩为一句金句

【CTA】"你怎么看？留言区聊聊" / 转发引导
```

#### 模板 C：情感共鸣型

```
【标题】情绪共鸣 或 身份代入

【开头】场景描写（感官细节，拉入情境）

【正文】
  ■ 故事展开（人物+事件+冲突）
  ■ 情绪递进（从平静到波动）
  ■ 高潮/转折（最触动人心的部分）
  ■ 感悟/升华（从个体经历提炼普遍意义）

【结尾】留白 或 一句余韵金句

【CTA】"转发给你想对TA说的人" / 在看引导
```

#### 模板 D：热点解读型

```
【标题】热点嫁接 + 独特角度

【开头】1-2句交代热点事件（假设读者已知背景）

【正文】
  ■ 大家都在关注的角度（简述，不赘述）
  ■ 我发现的不同角度（核心差异化）
  ■ 深度分析（因果推演/类比/数据支撑）
  ■ 对普通人的启示/影响

【结尾】回到个人/读者视角的思考

【CTA】转发引导（热点文章天然有传播动力）
```

#### 模板 E：故事叙事型

```
【标题】悬念留白 或 对比冲突

【开头】从一个具体时刻切入（不要从"故事背景"开始）

【正文】
  ■ 主人公的困境/起点
  ■ 转折事件（改变轨迹的关键节点）
  ■ 行动与挣扎（展现过程，不是结果概述）
  ■ 结果与现状

【结尾】一句话点睛（不过度解读，让故事本身说话）

【CTA】"你身边有类似经历的人吗？" / 关注引导
```

#### 模板 F：盘点清单型

```
【标题】清单诱惑 + 数字

【开头】说明筛选标准和信息来源（建立可信度）

【正文】
  ■ 推荐 1：名称 + 一句话亮点 + 适用场景 + 使用感受
  ■ 推荐 2：...
  ■ ...
  ■ 推荐 N：...
  ■ 对比总结表格（可选）

【结尾】"收藏这篇，下次用到的时候翻出来"

【CTA】收藏引导 + "还有什么好用的补充？留言区见"
```

## 七、开头钩子技巧（Hook Techniques）

公众号文章的开头 3 句话（约 50-80 字）决定读者是否继续阅读。

### 7.1 六种高完读率开头

| 钩子类型 | 原理 | 示例 |
|---------|------|------|
| 痛点场景 | 唤醒读者的真实困扰 | "每天加班到9点，工资卡的数字还是没什么变化。你有没有想过，问题出在哪里？" |
| 数据冲击 | 用反直觉的数据抓注意力 | "据统计，90%的公众号文章在3秒内就被划走了。如果你还在看，说明这个标题做对了一件事。" |
| 故事悬念 | 开头即冲突，不揭底 | "去年，我花了2万块报了一个写作课。三个月后，我退了款，但赚回了10倍。" |
| 读者对话 | 直接与读者建立联系 | "在开始之前，我想先问你一个问题：你上次认认真真读完一篇3000字的文章，是什么时候？" |
| 热点切入 | 借势当下关注焦点 | "[某事件]这两天刷屏了。所有人都在讨论[表面话题]，但我想聊点不一样的。" |
| 结果前置 | 先展示结果，激发好奇 | "上个月，我用一个方法把公众号涨了3000粉。今天把方法全部写出来，你可以直接抄。" |

### 7.2 开头禁忌

```
✗ 大段引用/名人名言开头 → 读者没耐心看别人说的话
✗ "今天来聊一个话题…" → 毫无信息量
✗ 长篇背景介绍 → 读者不关心你的铺垫
✗ "最近很多人问我…" → 这是自嗨，不是读者需求
✗ 直接放广告/引导关注 → 还没给价值就索取
```

## 八、结尾与收束技巧（Closing Strategies）

### 8.1 四种结尾方式

| 结尾方式 | 适用类型 | 效果 | 示例 |
|---------|---------|------|------|
| 金句收束 | 观点/情感型 | 被截图传播 | "真正的成长不是学会了多少道理，而是终于开始做了。" |
| 行动号召 | 干货教程型 | 促进实践 | "现在就打开你的[工具]，把第一步做了。别收藏了不看——从今天开始。" |
| 引发思考 | 观点/热点型 | 引导留言互动 | "这件事没有标准答案。但如果你是当事人，你会怎么选？" |
| 情感留白 | 情感/故事型 | 余韵悠长 | "那天的夕阳很好看。我后来再也没去过那条路。" |

### 8.2 结尾禁忌

```
✗ "以上就是今天的分享" → 多余，打断情绪
✗ "希望对你有所帮助" → 敷衍、模板感
✗ "喜欢就点个在看+转发+关注吧" → 三连要求 = 读者什么都不做
✗ 重复标题内容 → 除非刻意首尾呼应
✗ 突然硬切广告 → 前面写了2000字好内容，结尾全毁
```

## 九、Agent 行为指南

当用户要求写公众号文章时，按以下流程执行：

### 9.1 执行流程

1. **确认需求**：主题、调性、目标读者、文章目的、字数要求
2. **判定文章类型**：匹配六大类型中最合适的
3. **选择结构模板**：根据类型选择对应模板（A-F）
4. **撰写标题**：根据文章类型选择标题公式，至少提供 2 个备选标题
5. **撰写开头**：根据类型选择钩子技巧
6. **撰写主体**：按结构模板展开内容
7. **撰写结尾**：选择合适的收束方式
8. **设计CTA**：根据文章目的配置 CTA
9. **排版优化**：根据调性调整排版参数
10. **质量检查**：逐项核对检查清单

### 9.2 输出格式

```
## 文章分析
- 文章类型：[类型]
- 推荐结构：[模板编号+名称]
- 目标调性：[调性类型]
- 主CTA：[CTA类型+文案]

## 备选标题
1. [标题A]（使用公式：XX）
2. [标题B]（使用公式：XX）
→ 推荐标题：[编号]，理由：[一句话]

---

## 正文

[完整的公众号文章内容，含排版标注]

---

## 排版建议
- 字号/间距/颜色建议
- 配图建议（位置+风格）
- 加粗位置标注

## 写作说明
- 结构选择理由
- 金句位置标注
- CTA设计逻辑
- 可调整方向建议
```

### 9.3 质量检查清单

文章生成后逐项检查：

- [ ] 标题是否在13字内传达核心卖点？
- [ ] 开头3句话是否能留住读者？
- [ ] 每段是否控制在手机4-5行以内？
- [ ] 加粗是否只用在关键观点（不超过全文5%）？
- [ ] 每屏是否有视觉锚点（加粗/图片/小标题/引用）？
- [ ] 文章调性是否全文一致？
- [ ] CTA是否不超过3处？主CTA是否只有1个？
- [ ] CTA是否与正文自然衔接？
- [ ] 是否有至少1个可截图传播的金句？
- [ ] 结尾是否有力，不拖沓？
- [ ] 标题与正文内容是否一致（不做标题党欺诈）？
- [ ] 排版参数是否符合调性？
