# 风格规范：大宗商品研究展望

基于机构卖方"大宗商品观点"系列（2024、2025、2026版）提炼。

**主要风格参考：2026版**——简洁极简白色背景设计。早期2024/2025版封面使用深色深蓝渐变条（已弃用的风格）。

## 目录

1. 页面构成与版面
2. 字体排版系统

4. 图表与展示处理
5. 页眉、页脚与导航
6. 封面页设计

## 1. 页面构成与版面

### 页面尺寸
US Letter（8.5 x 11英寸），纵向排版。

### 页边距
- 上：约0.6英寸（为页眉预留空间）
- 下：约0.7英寸（为日期和页码预留空间）
- 左：约0.8英寸
- 右：约0.8英寸

### 正文版面
- 单栏文本流
- 图表内联嵌入（不与文本并排）
- 多面板图表：在单个图表展示中将两个图表并排放置
- 图表展示时可占页面高度的40-60%

### 内容密度
- 中高密度：包含分析段落的研究文档
- 每页：3-5个段落 + 0-1个图表
- 图表密集的页面文字相应减少

## 2. 字体排版系统

### 参考字体（来自PDF字体提取）

**2026版（当前主要风格）：**
- **正文/标题**：几何新怪诞无衬线字体族（Regular、Bold、Medium、Light、Italic、Condensed-Regular等字重）
- **图表标签**：ArialMT、Arial-BoldMT
- **辅助字体**：UniversLTStd-Light、RobotoCondensed-Regular

**2024/2025版（旧版风格——非目标风格）：**
- UniversLTStd-BoldCn、UniversLTStd-Cn、UniversLTStd-LightCn、UniversLTStd-Light

### 字体排版特征
- **类别**：几何新怪诞无衬线体或窄体怪诞无衬线体（UniversLTStd）
- **正式程度**：专业、机构级、简洁
- **字重对比**：中等——标题使用粗体，正文使用常规体，次要元素使用细体
- **密度**：适中，紧凑版面使用窄体变体

### 回退策略（关键——禁止使用LiberationSans）

| 用途 | 参考字体 | 不可用时的替代 |
|---|---|---|
| 所有标题 | Helvetica Neue Bold / Arial Bold | — |
| 正文文本 | Helvetica Neue / Arial Regular | — |
| 细体/次要文本 | Helvetica Neue Light | Arial Light |
| 窄体标题 | Arial Narrow / Helvetica Neue Condensed | — |
| 图表标签 | Arial-BoldMT / ArialMT | Arial Bold / Arial Regular |

**关键规则**：在整个文档中一致使用几何无衬线体（Helvetica Neue、Arial）。视觉特征应简洁、现代、具有机构感——而非LiberationSans的独特外观。

### 字号层级（视觉估算）

| 元素 | 字号 | 字重 |
|---|---|---|
| 封面标题 | 20-22pt | 粗体 |
| 系列标签（"大宗商品观点"） | 11-12pt | 粗体，大写/小型大写 |
| 主题/小节标题（内联） | 13-14pt | 粗体 |
| 正文文本 | 10-11pt | 常规 |
| 图表标题 | 10-11pt | 粗体 |
| 来源标注 | 7-8pt | 常规 |
| 页眉文本 | 8-9pt | 常规 |
| 脚注 | 7-8pt | 常规 |

### 文本对齐
- 正文文本：左对齐，右侧不齐
- 标题/小标题：左对齐，与文本流内联
- 图表标题：左对齐

### 行距
- 正文：约1.4-1.5倍行距
- 段间距：段后约6-8pt
- 章节间距：新主题开始时留较大间距（约12-16pt）

## 3. 色彩方案

### 品牌与页面颜色

| 用途 | 颜色 | 色值 |
|---|---|---|
| 正文文本 | 极深灰/黑色 | #1A1A1A |
| 主要强调色（链接、交叉引用） | 深蓝 | #003366 |
| 次要标题强调色 | 中灰蓝色 | #4A6B8A |
| 页面背景 | 白色 | #FFFFFF |
| 次要文本/来源标注 | 深灰色 | #555555 |
| 竖线装饰线（封面左侧边距） | 深蓝 | #003366 |
| 页脚分隔线 | 浅灰色 | #CCCCCC |

### 图表配色

| 系列 | 颜色 | 色值 | 用途 |
|---|---|---|---|
| 主系列 | 深蓝 | #003366 | 主要线条/系列 |
| 次要/对比 | 红色/陶土色 | #C62828 | 对比、负面、相反方向 |
| 上涨/正面 | 森林绿 | #2E7D32 | 正面趋势 |
| 警示/中性 | 金色/琥珀色 | #F9A825 | 中性/警示信号 |
| 第三系列/区域 | 各种柔和色调 | — | 额外数据系列 |
| 图表网格线 | 极浅灰色 | #E0E0E0 | 可选，仅水平方向 |

### 强调元素使用
- 不使用彩色章节标题栏（2026版风格）
- 超链接交叉引用：蓝色下划线
- 核心要点：实心黑色方块（主要项目符号），空心方块（子项目符号）
- 顶级交易建议：空心方块复选框
- 粗体内联小标题：黑色文本，无背景色

## 4. 图表与展示处理

### 图表命名
- 格式：**"图表 N：[描述性标题]"**——粗体
- 全文按顺序编号
- 长描述允许使用多行标题
- 正文中交叉引用："（图表 N）"或"图表 N，左侧面板"

### 图表设计规则
- **背景**：白色，无填充
- **网格线**：仅极浅灰色水平网格线（可选），尽量简洁
- **坐标轴**：黑色轴线，无粗边框（开放式坐标轴风格）
- **标签**：清晰，位于图表区域内部或相邻位置
- **图例**：位于顶部或右侧，紧凑排列，彩色方块/条形配标签
- **标注**：关键事件直接标注在图表上，配箭头/线条（如"本轮周期首次降息"、"冻结俄罗斯央行资产"）
- **双轴**：常见——左轴一个指标，右轴另一个指标（不同单位）
- **填充样式**：实心深蓝、实心红色，次要填充使用斜线/对角线图案

### 常见图表类型
1. **双面板时间序列**——两个相关的折线图/柱状图并排展示
2. **堆叠柱状图**——按时间展示成分分解
3. **分组柱状图**——区域/商品对比，使用深蓝和红色
4. **回归散点图**——配趋势线、R2标注、预测点圈标
5. **框架示意图**——环形/流程图风格（如"大宗商品控制周期"）
6. **数据表格**——结构化预测表格，深蓝色表头行

### 表格设计（图表中的表格）
- 表头行：深蓝色背景（#003366），白色粗体文字
- 数据行：白色/极浅灰色交替（可选）
- 数字列右对齐
- 关键汇总行使用粗体（如"宏观交易"、"微观交易"）
- 使用水平线分隔各部分

### 来源标注
每个图表均包含：**"来源：[数据提供方]，[机构名称]全球投资研究"**
- 位置：图表下方，左对齐
- 字号：小号（等效7-8pt）
- 颜色：灰色（#555555）

## 5. 页眉、页脚与导航

### 页眉（仅正文页）
- 左侧："[机构名称]"，小号文字（约8-9pt）
- 右侧："大宗商品观点"，小号文字（约8-9pt）
- 文字下方以细水平线分隔
- 封面页无页眉

### 页脚
- 左下：日期（如"2025年12月18日"）
- 右下：页码
- 分隔方式：正文页页脚内容上方有细水平线

### 封面页边缘文字
- 左侧边缘（竖排）："仅供[客户名称]专用"——小号文字，旋转90度，灰色

## 6. 封面页设计（2026+风格）

### 关键：禁止深色渐变条

2026版使用完全干净的白色背景封面。早期2024/2025版使用深色深蓝渐变条——这不是目标风格。

### 版面布局（从上到下）

1. **品牌页眉**（左上）：
   - 机构名称采用堆叠文字标识风格（两行排列）
   - 堆叠标识旁标注"研究"或"大宗商品研究"
   - 白色背景上仅使用纯文本

2. **日期/时间**（右上）：
   - "大宗商品研究"使用粗体/强调文字
   - 下方显示："2025年12月18日 | 11:33AM EST"

3. **竖线装饰线**（左侧边距）：
   - 品牌页眉下方的细竖线深蓝色装饰线
   - 沿左侧延伸部分长度

4. **系列标签**："大宗商品观点"（粗体，较小字号，全部大写或小型大写）

5. **报告标题**：大号粗体标题（如"2026年展望：把握能源竞赛与供应浪潮"）

6. **水平分隔线**：将页眉区域与内容区域分隔的细灰线

7. **分隔线下方双栏布局**：
   - **左栏**（约60%宽度）：5-8条核心要点，使用实心方块标记
     - 主要项目符号：大实心方块
     - 子项目符号：空心方块，配粗体交易名称（如"**我们做多黄金的交易**建议……"）
   - **右栏**（约40%宽度）：作者团队列表
     - 每位分析师：**姓名** | 电话 | 邮箱 | 所属机构

8. **顶级交易建议部分**（双栏区域下方，占据全宽）：
   - "顶级交易建议："配实心方块
   - 3-4个交易建议，使用空心方块复选框，粗体交易名称，简要描述

9. **免责声明栏**（底部）：
   - 细线分隔
   - 小号监管文字："投资者应仅将本报告视为投资决策的众多参考因素之一……"
   - 指向披露附录的链接

## 7. 正文页设计

### 首个正文页（第2页）
- 显示页眉
- 顶部重复报告标题："2026年展望：把握能源竞赛与供应浪潮"
- 开篇论点段落（4-6句话）
- 核心主题要点概览
- 首批图表内联出现（图表1、图表2等）

### 典型正文页流程
- 顶部页眉
- 文本段落配粗体内联小标题标记新主题
- 图表出现在段落之间（不单独分节）
- 脚注出现在页面底部，使用上标编号
- 页脚显示日期和页码

### 正文中的小标题风格
- 段落开头或独立行的粗体文本
- 参考示例："**做多黄金**"、"**大宗商品的保险价值**"、"**做多铜，做空铝**"，下方配斜体描述行（如"*2026年铜价在关税利好消化后趋于盘整*"）
- 紧跟分析段落

### 交叉引用风格
- 图表引用：下划线和/或蓝色"（图表 N）"或"图表 N，左/右面板"
- 外部链接：带超链接样式的下划线文本
- 此前预测对比：正文中内联显示"相比此前 $98"
