## 1. 文档章节层级

```
Cover Page (page 1)
|-- Brand text header (NO gradient bars)
|-- Vertical navy left-accent line
|-- Series label + title
|-- Key Takeaways (left, 60%): 5-8 bullets with filled-square markers
|-- Author Team (right, 40%): Name | Phone | Email | Entity
|-- Top Trades: 3-4 entries with empty-square checkboxes
|-- Disclaimer bar at bottom

Body Pages (pages 2 to N-1) — NO Table of Contents page
|-- Running header ("[Firm Name]" | "Commodity Views")
|-- Opening thesis narrative + overview bullets
|-- Exhibit 1: Initial multi-panel chart or trade recommendations table
|-- Theme 1: Bold inline heading + thesis paragraph + exhibits
|-- Theme 2: Bold inline heading + thesis paragraph + exhibits
|-- Theme 3: Bold inline heading + thesis paragraph + exhibits
|-- Per-commodity deep dives (oil, gas, metals, power)
|-- ~15-16 total exhibits across ~17-18 body pages

Disclosure Appendix (last 1-2 pages)
|-- Global product / distributing entities
|-- Country-specific regulatory notices (Japan, EU, etc.)
|-- General disclosures (clients only, forward-looking statements)
|-- Differing levels of service
|-- Copyright + redistribution restrictions
```

**总页数**：约19-26页，视内容深度而定
**图表密度**：约19页报告中包含15-16个图表，约每页0.8-1.0个图表

## 2. 封面页元素

### 必要元素

| 元素 | 内容规则 |
|---|---|
| **报告标题** | "[年份]展望：[副标题]"——投资主题副标题 |
| **日期** | 发布日期和时间戳 |
| **系列名称** | "大宗商品观点" |
| **核心要点** | 5-8条。每条为一个明确的方向性判断，包含：(a) 方向性建议，(b) 预测/目标价，(c) 关键驱动因素。主要项目符号使用实心方块；子项目符号使用空心方块配粗体交易名称。 |
| **作者团队** | 8-12位分析师。每位：**姓名** \| 电话 \| 邮箱 \| 所属机构 |
| **顶级交易建议** | 3-4个命名交易。格式：空心方块 **交易名称**：简要描述 |

### 封面页内容模式

核心要点项目结构：
```
- Directional thesis statement (e.g., "The US-China AI and geopolitical power race...")
  Sub-bullets with bold trade names:
  - "Our long gold trade recommendation as EM central banks diversify..."
  - "The insurance value of commodities in portfolios as supply concentration grows..."
```

顶级交易建议格式：
```
Top trades:
  ☐  Trade Name: Brief description (1-2 lines)
  ☐  Trade Name: Brief description
```

## 3. 正文页流程

### 首个正文页（第2页）——开篇叙述

1. **顶部重复报告标题**（与封面标题相同）
2. **论点段落**（4-8句话）：总体观点、商品指数背景、方向性判断
   - 示例："2025年大宗商品指数取得了强劲的总回报……"
   - 引用宏观基准情景，附超链接风格的交叉引用
3. **核心主题要点概览**：使用实心方块标记列举主要主题
   - 每条将主题与具体交易建议关联
   - 交叉引用图表："（图表1，左侧面板）"
4. **首批图表**：通常在靠前页面出现多面板图表或交易建议表格
5. **后续叙述**：文本流畅过渡到第二个主题领域

### 后续正文页

- **无正式分节**——内容连续流动
- **粗体内联小标题**（如"**做多黄金**"）标记新的分析主题
- **斜体描述性副标题**可紧随其后（如"*2026年价格在最后一轮大供应浪潮中下行*"）
- **图表嵌入段落之间**
- **脚注**出现在页面底部，使用上标编号
- **交易建议**在正文中以粗体形式表述（如"我们继续**建议做多黄金**。"）

## 4. 主题章节模式

### 章节标题风格（内联，不编号）

章节使用粗体内联标题，而非正式的章节划分：

```
**The Geopolitical Power and AI Race Between the US and China**

**Long Gold**
The gold rally accelerated in 2025 (up 64% YTD) as falling US interest rates led ETF 
investors... We expect the same two drivers—structurally high central bank demand and 
cyclical support from Fed cuts—to lift the gold price further to $4,900/toz by Dec26. 
We thus continue to recommend long exposure in gold.

[Exhibit: multi-panel chart showing gold price and demand]
```

### 每个主题内的分析逻辑

```
1. Establish macro context → link to top-down narrative
2. Supply-side evidence → exhibits with inventory/capacity data
3. Demand-side evidence → exhibits with consumption/flow data  
4. Price forecast → explicit target levels and timeframes
5. Trade recommendation → bold statement in text + risk qualifiers
6. Cross-reference to prior/future exhibits for continuity
```

### 小标题序列示例（来自2026版参考）

- "中美之间的地缘政治与AI竞赛"
  - "**做多黄金**"
  - "**大宗商品的保险价值**"
  - "**做多铜，做空铝**"
  - "**我们对美国电力市场趋紧的判断**"
- "把握能源供应浪潮"
  - "**2026年原油供过于求**" / "*2026年价格在最后一轮大供应浪潮中下行*"
  - "**天然气：欧洲和亚洲的延迟再平衡**"

## 5. 图表系统

### 编号规则
- 全文按顺序编号：图表1、2、3……
- 不使用章节前缀
- 交叉引用："（图表 N）"或"（图表 N，左/右面板）"

### 图表类型与位置

| 类型 | 描述 | 典型位置 |
|---|---|---|
| 多面板柱状图 | 两个分组柱状图并排 | 靠前页面（覆盖主题） |
| 双轴时间序列 | 带两个Y轴的折线图 | 商品价格趋势 |
| 交易建议表格 | 深蓝色表头，粗体交易名称，描述+理由 | 图表2-3 |
| 回归散点图 | 配趋势线、R2、高亮预测点 | 价格预测验证 |
| 框架示意图 | 环形/流程图（如大宗商品控制周期） | 概念性章节 |
| 堆叠/分组柱状图 | 供给集中度、市场份额 | 供给分析 |
| 数据表格 | 结构化数字预测 | 预测汇总 |

### 多面板图表解剖
```
Exhibit N: [Descriptive Multi-Clause Title]

[Left chart title]                    [Right chart title]
[Chart with axis labels, legend]      [Chart with axis labels, legend]
                                      [Annotation: "Nov25: US Level = 44.6 GW"]

Source: [Data Providers], [Firm Name] Global Investment Research
```

### 图表脚注
- 部分复杂图表在图表下方包含技术脚注
- 格式：较小字号，解释方法论、定义、排除项

## 6. 交叉引用系统

| 引用类型 | 格式 | 示例 |
|---|---|---|
| 图表 | 带下划线，括号内 | "对冲地缘政治/制裁风险（图表1，左侧面板）" |
| 超链接 | 下划线文本 | "美联储降息"、"中美竞赛"（链接样式） |
| 脚注 | 上标数字 | "……如稀土所发生的情况。²" |
| 此前预测 | 正文内联对比 | "相比此前 $98"、"（相比此前 $12,000/吨）" |

---

## 7. 脚注

- 使用上标按顺序编号
- 置于引用所在页面底部
- 与正文以细水平线分隔
- 使用小号字体
- 内容：方法论说明、假设条件、排除项、数据注意事项

示例：
```
2  China's exports restrictions remain in place for the geologically scarce so-called heavy rare earths.
```

## 8. 披露附录

### 必要章节

1. **各国监管公告**——日本、欧盟/英国、巴西等
   - 机构注册信息、监管机构、联系方式
   
2. **全球产品；分销机构**——研究报告分销机构完整列表

3. **一般性披露**——多段标准法律文本：
   - 本研究仅供客户使用
   - 基于被认为可靠的当前公开信息
   - 观点可能随时变更，恕不另行通知
   - 本文件不构成要约或招揽
   - 过往表现不代表未来业绩
   - 衍生品风险披露

4. **差异化服务水平**——全球投资研究服务水平可能因地区而异

5. **再分发/AI使用限制**（2025+版本）：
   - "未经许可，您不得发布、传输或以其他方式复制本信息……"
   - "……亦不得将本信息作为提示或输入提供给任何大语言模型系统。"

6. **版权**：
   - "(C) [年份] [机构名称]。"
   - 具体使用权限及限制

### 披露格式
- 字号小于正文
- 全宽文本块
- 仍保留页眉
- 页码延续
