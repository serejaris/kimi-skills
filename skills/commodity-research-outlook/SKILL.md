# 大宗商品研究展望

按照经过验证的卖方研究结构，制作机构级大宗商品展望报告。

## 参考来源

- **类型**：上传的PDF文件（三个版本：2024、2025、2026）
- **参考文件类型**：PDF
- **主要风格参考**：2026版（简洁极简设计，几何无衬线字体排版）
- **支持的输出格式**：PDF（主要）、DOCX（可编辑）、PPTX（演示文稿子集）
- **默认输出**：用户未指定时默认输出PDF

## 风格规范

加载 `references/style_contract.md` 获取完整视觉规范。关键规则如下。

### 必须遵循的风格规则

1. **封面页——禁止使用深色渐变条**。全页白色背景。纯文本页眉：品牌Logo名称位于左上方，"大宗商品研究"位于右上方并附日期/时间。页眉下方左侧边距处有细竖线深蓝色装饰线。
2. **不使用章节编号层级**（不使用1.、1.1、2.等）。采用段落式流畅文本，配合粗体小标题（如"做多黄金"或"大宗商品的保险价值"）嵌入叙述中。
3. **不使用彩色章节标题栏**。所有正文页均为干净的白色背景。
4. **字体排版**：几何新怪诞无衬线体；使用Helvetica Neue / Arial作为主要字体。禁止使用LiberationSans。
5. **图表**：深蓝色（#003366）和红色/陶土色（#C62828）作为主要系列颜色。简洁极简网格线。双面板图表（两个图表并排）为标准展示方式。

### 快速参考

## 结构规范

加载 `references/structure_contract.md` 获取完整结构规范。

### 文档流程（无目录页）

```
Cover Page (page 1)
  |-- Brand text header + date
  |-- Thin vertical navy left-accent line
  |-- "COMMODITY VIEWS" series label
  |-- Report title + subtitle
  |-- Key takeaways (left column, ~60%): 5-8 bullets with filled-square markers
  |-- Author team (right column, ~40%): Name | Phone | Email | Entity
  |-- Top trades: empty-square markers with bold trade names
  |-- disclaimer bar at bottom
  
Body Pages (page 2 onward) — NO TOC PAGE, flow directly into content
  |-- Running header + thin rule
  |-- Opening thesis paragraph + overview narrative
  |-- Exhibits inline with text (multi-panel charts, tables)
  |-- Thematic sections with bold subheadings in text flow
  |-- Per-commodity deep dives with supply/demand analysis
  |-- ~15-16 exhibits per 19-page report (high exhibit density)

Disclosure Appendix (last 1-2 pages)
  |-- Reg AC certification
  |-- Global product / distributing entities
  |-- General disclosures
  |-- Copyright notice
```

## 图表要求

- **数量**：约19页的报告目标包含15-16个实质性图表。每个主要主题需要2-4个配套图表。
- **类型**：多面板时间序列（双轴）、分组/堆叠柱状图、回归散点图、数据表格、框架示意图。
- **交叉引用**：每个图表在正文中以"（图表 N）"或"图表 N，左侧面板"形式引用。
- **来源标注**：每个图表均需标注数据来源。

## 工作流程

### 第1步：收集输入信息

- 展望的目标年份/时间段
- 需覆盖的大宗商品板块（能源、金属、农产品）
- 关键宏观假设（GDP增长、美联储政策、地缘政治背景）
- 需要重点强调的具体交易观点或方向性判断
- 作者/团队署名

### 第2步：构建结构

1. 根据宏观主题确定2-4个核心主题
2. 将各大宗商品板块对应到相关主题
3. 为封面页起草5-8条核心要点
4. 编制3-4个顶级交易建议，使用空心方块标记
5. 规划约15-16个图表及数据来源

### 第3步：起草内容

对每个主题，遵循以下叙述模式：
- **开篇论点** → 1-2段概述
- **宏观背景** → 关联自上而下的分析逻辑，交叉引用图表
- **供给侧分析** → 产量、库存、剩余产能
- **需求侧分析** → 各区域需求、结构性与周期性驱动因素
- **价格预测** → 基准情景及多空情景敏感性分析
- **交易建议** → 在文本中嵌入粗体建议声明

### 第4步：审核清单

- [ ] 封面页为纯文本页眉（禁止渐变条），左侧有竖线装饰线
- [ ] 无目录页——直接从封面过渡到正文
- [ ] 无编号章节层级——仅使用粗体内联小标题
- [ ] 每个主题有2-4个配套图表
- [ ] 所有约15-16个图表按顺序编号并交叉引用
- [ ] 每个图表均有来源标注
- [ ] 交易建议包含粗体嵌入式声明和风险因素
- [ ] 明确阐述牛市/熊市/尾部情景风险
- [ ] 详细的披露附录，包含Reg AC认证、监管公告、版权声明
- [ ] 字体为Helvetica Neue / Arial（或同类几何无衬线体）——禁止使用LiberationSans
