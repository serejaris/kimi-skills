# SaaS 指标公式

SaaS 指标教练所计算的所有指标的完整公式参考及计算示例。

## ARR（年度经常性收入）
```
ARR = MRR × 12
```

**示例：**
- 当前 MRR：$50,000
- ARR = $50,000 × 12 = **$600,000**

**使用场景：** 快速了解年化收入运行速率。如果存在季节性波动或一次性费用，ARR 不等于实际年收入。

## MoM MRR 月环比增长率
```
月环比增长率 % = ((当前 MRR - 上月 MRR) / 上月 MRR) × 100
```

**示例：**
- 当前 MRR：$50,000
- 上月 MRR：$45,000
- 增长率 = (($50,000 - $45,000) / $45,000) × 100 = **11.1%**

**解读：**
- 负值 = 收入在减少
- 0-5% = 增长缓慢（早期阶段需警惕）
- 5-15% = 健康增长
- >15% = 强劲增长（早期阶段）

## 月流失率
```
流失率 % = (本月流失客户数 / 月初客户总数) × 100
```

**示例：**
- 月初客户数：100
- 本月流失客户数：5
- 流失率 = (5 / 100) × 100 = **5%**

**年化影响：** 月流失率 5% ≈ 年流失率约 46%（复利效应）

**重要提示：** 不同市场细分对流失率的容忍度不同：
- 企业级：>3% 为危急
- 中小企业：>8% 为危急
- 给出判断前务必确认市场细分

## ARPA（客均收入）
```
ARPA = MRR / 活跃客户总数
```

## CAC（客户获取成本）
```
CAC = 销售与营销总支出 / 新获取客户数
```
示例：$20k 支出 / 10 个客户 → CAC $2,000

## LTV（客户生命周期价值）
```
LTV = (ARPA / 月流失率) × 毛利率 %
```

**简化版（无毛利率数据时）：**
```
LTV = ARPA / 月流失率
```

**示例：**
- ARPA：$500
- 月流失率：5%（0.05）
- 毛利率：70%（0.70）
- LTV = ($500 / 0.05) × 0.70 = **$7,000**

**简化版（无毛利率数据）：** $500 / 0.05 = **$10,000**

**为什么重要：** LTV 代表一个普通客户在其整个生命周期内预期带来的总收入。LTV 至少应为 CAC 的 3 倍，才能保证单位经济模型的可持续性。

## LTV:CAC 比率
```
LTV:CAC = LTV / CAC
```
示例：LTV $10k / CAC $2k = 5:1

## CAC 回本周期
```
回本周期（月）= CAC / (ARPA × 毛利率 %)
简化版：回本周期 = CAC / ARPA
```
示例：CAC $2k / ARPA $500 = 4 个月

## NRR（净收入留存率）
```
NRR % = ((期初 MRR + 扩展 MRR - 流失 MRR - 收缩 MRR) / 期初 MRR) × 100
```
简化版（无扩展数据时）：NRR ≈ (1 - 收入流失率) × 100

## 40 法则
```
得分 = 年化月环比增长率 % + 净利润率 %
健康：≥ 40
```
