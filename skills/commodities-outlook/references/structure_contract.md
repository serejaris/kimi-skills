## 1. Document Section Hierarchy

```
Cover Page (page 1)
|-- Brand text header (NO gradient bars)
|-- Vertical navy left-accent line
|-- Series label + title
|-- Key Takeaways (left, 60%): 5-8 bullets with filled-square markers
|-- Author Team (right, 40%): Name | Phone | Email | Entity
|-- Top Trades: 3-4 entries with empty-square checkboxes
|-- Disclaimer bar at bottom

Body Pages (pages 2 to N-1) — NO Table of Contents page
|-- Running header ("[Firm Name]" | "Commodity Views")
|-- Opening thesis narrative + overview bullets
|-- Exhibit 1: Initial multi-panel chart or trade recommendations table
|-- Theme 1: Bold inline heading + thesis paragraph + exhibits
|-- Theme 2: Bold inline heading + thesis paragraph + exhibits
|-- Theme 3: Bold inline heading + thesis paragraph + exhibits
|-- Per-commodity deep dives (oil, gas, metals, power)
|-- ~15-16 total exhibits across ~17-18 body pages

Disclosure Appendix (last 1-2 pages)
|-- Global product / distributing entities
|-- Country-specific regulatory notices (Japan, EU, etc.)
|-- General disclosures (clients only, forward-looking statements)
|-- Differing levels of service
|-- Copyright + redistribution restrictions
```

**Total pages**: ~19-26 pages depending on content depth
**Exhibit density**: ~15-16 exhibits per 19-page report, ~0.8-1.0 exhibits per page

## 2. Cover Page Elements

### Required Elements

| Element | Content Rules |
|---|---|
| **Report Title** | "[Year] Outlook: [Subtitle]" — investment thesis subtitle |
| **Date** | Publication date and time stamp |
| **Series Name** | "Commodity Views" |
| **Key Takeaways** | 5-8 bullets. Each is a distinct conviction call with: (a) directional recommendation, (b) forecast/price target, (c) key driver. Main bullets use filled squares; sub-bullets use empty squares with bold trade names. |
| **Author Team** | 8-12 analysts. Each: **Name** \| Phone \| Email \| Entity |
| **Top Trades** | 3-4 named trades. Format: empty-square **Trade Name**: Brief description |

### Cover Page Content Pattern

Key takeaway bullet structure:
```
- Directional thesis statement (e.g., "The US-China AI and geopolitical power race...")
  Sub-bullets with bold trade names:
  - "Our long gold trade recommendation as EM central banks diversify..."
  - "The insurance value of commodities in portfolios as supply concentration grows..."
```

Top trades format:
```
Top trades:
  ☐  Trade Name: Brief description (1-2 lines)
  ☐  Trade Name: Brief description
```

## 3. Body Page Flow

### First Body Page (page 2) — Opening Narrative

1. **Report title repeated** at top (same as cover title)
2. **Thesis paragraph** (4-8 sentences): Overarching view, commodity index context, directional bias
   - Example: "Commodity indices have delivered strong total returns in 2025..."
   - References macro base case with hyperlink-style cross-references
3. **Bullet-point theme overview**: Main themes enumerated with filled-square markers
   - Each bullet links a theme to specific trade recommendations
   - Cross-references to exhibits: "(Exhibit 1, left panel)"
4. **First exhibits**: Usually multi-panel charts or trade recommendation tables appear early
5. **Continuing narrative**: Text flows into second thematic area

### Subsequent Body Pages

- **No formal section breaks** — content flows continuously
- **Bold inline subheadings** (e.g., "**Long Gold**") mark new analytical topics
- **Italic descriptive subtitle** may follow (e.g., "*Prices Down in 2026 on Last Big Supply Wave*")
- **Exhibits embedded between paragraphs**
- **Footnotes** appear at bottom of pages with superscript numbering
- **Trade recommendations** stated in bold within text (e.g., "We continue to **recommend long exposure in gold**.")

## 4. Thematic Section Pattern

### Section Heading Style (Inline, Not Numbered)

Sections use bold inline headings rather than formal chapter divisions:

```
**The Geopolitical Power and AI Race Between the US and China**

**Long Gold**
The gold rally accelerated in 2025 (up 64% YTD) as falling US interest rates led ETF 
investors... We expect the same two drivers—structurally high central bank demand and 
cyclical support from Fed cuts—to lift the gold price further to $4,900/toz by Dec26. 
We thus continue to recommend long exposure in gold.

[Exhibit: multi-panel chart showing gold price and demand]
```

### Analytical Arc Within Each Theme

```
1. Establish macro context → link to top-down narrative
2. Supply-side evidence → exhibits with inventory/capacity data
3. Demand-side evidence → exhibits with consumption/flow data  
4. Price forecast → explicit target levels and timeframes
5. Trade recommendation → bold statement in text + risk qualifiers
6. Cross-reference to prior/future exhibits for continuity
```

### Example Subheading Sequence (from 2026 reference)

- "The Geopolitical Power and AI Race Between the US and China"
  - "**Long Gold**"
  - "**Insurance Value of Commodities**"
  - "**Long Copper, Short Aluminum**"
  - "**Our US Power Market Tightening Call**"
- "Riding Energy Supply Waves"
  - "**2026 Oil Surplus**" / "*Prices Down in 2026 on Last Big Supply Wave*"
  - "**Natural Gas: A Delayed Rebalancing in Europe and Asia**"

## 5. Exhibit System

### Numbering
- Sequential: Exhibit 1, 2, 3, ... throughout the report
- No chapter-based prefixing
- Cross-reference: "(Exhibit N)" or "(Exhibit N, left/right panel)"

### Exhibit Types and Placement

| Type | Description | Typical Placement |
|---|---|---|
| Multi-panel bar chart | Two grouped bar charts side by side | Early pages (cover themes) |
| Dual-axis time-series | Line chart with two Y-axes | Commodity price trends |
| Trade recommendations table | Navy header, bold trade names, description + rationale | Exhibit 2-3 |
| Regression scatter plot | With trend line, R2, forecast points highlighted | Price forecast validation |
| Framework diagram | Circular/flow chart (e.g., commodity control cycle) | Conceptual sections |
| Stacked/grouped bar chart | Supply concentration, market share | Supply analysis |
| Data table | Structured numeric forecasts | Forecast summaries |

### Multi-Panel Exhibit Anatomy
```
Exhibit N: [Descriptive Multi-Clause Title]

[Left chart title]                    [Right chart title]
[Chart with axis labels, legend]      [Chart with axis labels, legend]
                                      [Annotation: "Nov25: US Level = 44.6 GW"]

Source: [Data Providers], [Firm Name] Global Investment Research
```

### Exhibit Footnotes
- Some complex exhibits include technical footnotes below the chart
- Format: smaller text, explaining methodology, definitions, exclusions

## 6. Cross-Reference System

| Reference Type | Format | Example |
|---|---|---|
| Exhibit | Underlined, in parentheses | "hedge geopolitical/sanctions risk (Exhibit 1, left panel)" |
| Hyperlink | Underlined text | "Fed cuts", "US-China race" (styled as links) |
| Footnote | Superscript number | "...as happened to rare earths.2" |
| Prior forecast | Inline comparison | "vs. $98 prior", "(vs. $12,000/t prior)" |

---

## 7. Footnotes

- Numbered sequentially with superscript
- Placed at bottom of the page where referenced
- Separated from body text by a thin horizontal rule
- Small font size
- Content: methodological notes, assumptions, exclusions, data caveats

Example:
```
2  China's exports restrictions remain in place for the geologically scarce so-called heavy rare earths.
```

## 8. Disclosure Appendix

### Required Sections

1. **Country-specific regulatory notices** — Japan, EU/UK, Brazil, etc.
   - Entity registrations, regulatory bodies, contact information
   
2. **Global product; distributing entities** — Full list of firm entities distributing research

3. **General disclosures** — Multi-paragraph standard legal text:
   - This research is for clients only
   - Based on current public information considered reliable
   - Opinions subject to change without notice
   - This is not an offer or solicitation
   - Past performance not indicative of future results
   - Derivatives risks disclosed

4. **Differing Levels of Service** — Global Investment Research service levels may vary

5. **Redistribution / AI restrictions** (2025+ editions):
   - "You are not permitted to publish, transmit, or otherwise reproduce this information..."
   - "...or to provide or reproduce this information as a prompt or input to any [AI] system."

6. **Copyright**:
   - "(C) [Year] [Firm Name]."
   - Specific usage permissions and restrictions

### Disclosure Format
- Smaller font than body text
- Full-width text blocks
- Running header still present
- Page numbers continue
