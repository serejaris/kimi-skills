# Style Contract: Commodity Research Outlook

Derived from institutional sell-side Commodity Views series (2024, 2025, 2026 editions).

**Primary style reference: 2026 edition** — clean minimal white-background design. Earlier 2024/2025 editions used dark navy gradient bars on cover (deprecated style).

## 1. Page Composition & Layout

### Page Size
US Letter (8.5 x 11 in), portrait orientation.

### Margins
- Top: ~0.6 in (room for running header)
- Bottom: ~0.7 in (room for date + page number)
- Left: ~0.8 in
- Right: ~0.8 in

### Body Layout
- Single-column text flow
- Exhibits embedded inline (not side-by-side with text)
- Multi-panel exhibits: two charts placed side-by-side within a single exhibit
- Exhibit may occupy 40-60% of page height when displayed

### Density
- Medium-high density: research document with analytical paragraphs
- Each page: 3-5 paragraphs + 0-1 exhibits
- Exhibit-rich pages may have less text

## 2. Typography System

### Reference Fonts (from PDF font extraction)

**2026 edition (current primary style):**
- **Body/headings**: Geometric neo-grotesque sans-serif family (Regular, Bold, Medium, Light, Italic, Condensed-Regular weights)
- **Chart labels**: ArialMT, Arial-BoldMT
- **Supplementary**: UniversLTStd-Light, RobotoCondensed-Regular

**2024/2025 editions (legacy style — NOT the target):**
- UniversLTStd-BoldCn, UniversLTStd-Cn, UniversLTStd-LightCn, UniversLTStd-Light

### Typography Characteristics
- **Category**: Geometric neo-grotesque sans-serif or condensed grotesque sans-serif (UniversLTStd)
- **Formality**: Professional, institutional, clean
- **Weight contrast**: Medium — Bold for headings, Regular for body, Light for subtle elements
- **Density**: Moderate, with condensed variants for tight layouts

### Fallback Strategy (CRITICAL — DO NOT use LiberationSans)

| Role | Reference | When unavailable |
|---|---|---|
| All headings | Helvetica Neue Bold / Arial Bold | — |
| Body text | Helvetica Neue / Arial Regular | — |
| Light/subtle text | Helvetica Neue Light | Arial Light |
| Condensed headers | Arial Narrow / Helvetica Neue Condensed | — |
| Chart labels | Arial-BoldMT / ArialMT | Arial Bold / Arial Regular |

**Key rule**: Use a geometric sans-serif (Helvetica Neue, Arial) consistently across the document. The visual character should be clean, modern, and institutional — not the distinctive look of LiberationSans.

### Size Hierarchy (visual estimate)

| Element | Size | Weight |
|---|---|---|
| Cover title | 20-22pt | Bold |
| Series label ("COMMODITY VIEWS") | 11-12pt | Bold, caps/small-caps |
| Theme/subsection heading (inline) | 13-14pt | Bold |
| Body text | 10-11pt | Regular |
| Exhibit title | 10-11pt | Bold |
| Source line | 7-8pt | Regular |
| Running header text | 8-9pt | Regular |
| Footnotes | 7-8pt | Regular |

### Text Alignment
- Body text: left-aligned, ragged right
- Headers/subheadings: left-aligned, inline with text flow
- Exhibit titles: left-aligned

### Line Spacing
- Body: ~1.4-1.5 line spacing
- Between paragraphs: ~6-8pt after
- Between sections: larger gap (~12-16pt) when new theme begins

## 4. Chart & Exhibit Treatment

### Exhibit Naming
- Format: **"Exhibit N: [Descriptive Title]"** — bold
- Numbered sequentially throughout the report
- Multi-line titles allowed for long descriptions
- Cross-referenced in body: "(Exhibit N)" or "Exhibit N, left panel"

### Chart Design Rules
- **Background**: White, no fill
- **Gridlines**: Very light gray horizontal gridlines only (optional), minimal
- **Axes**: Black axis lines, no bold framing box (open-axis style)
- **Labels**: Clear, inside or adjacent to chart area
- **Legend**: Top or right side, compact, colored boxes/bars with labels
- **Annotations**: Key events labeled directly on chart with arrows/lines (e.g., "First Fed Cut of Cycle", "Freezing of Russia Central Bank Assets")
- **Dual-axis**: Common — left axis for one metric, right axis for another (different units)
- **Fill patterns**: Solid navy, solid red, hatched/diagonal patterns for secondary fills

### Common Chart Types
1. **Dual-panel time-series** — Two related line/bar charts side by side
2. **Stacked bar** — Component breakdown over time
3. **Grouped bar** — Regional/commodity comparison with navy + red bars
4. **Regression scatter** — With trend line, R2 annotation, forecast points circled
5. **Framework diagram** — Circular/flowchart style (e.g., "Commodity Control Cycle")
6. **Data table** — Structured forecast tables with navy header row

### Table Design (Exhibit tables)
- Header row: navy background (#003366) with white bold text
- Body rows: alternating white / very light gray (optional)
- Right-aligned numeric columns
- Bold for key summary rows (e.g., "Macro Trades", "Micro Trades")
- Separating sections with horizontal rules

### Source Attribution
Every exhibit includes: **"Source: [Data Providers], [Firm Name] Global Investment Research"**
- Position: below exhibit, left-aligned
- Size: small (7-8pt equivalent)
- Color: gray (#555555)

## 5. Headers, Footers & Navigation

### Running Header (Body Pages Only)
- Left: "[Firm Name]" in small text (~8-9pt)
- Right: "Commodity Views" in small text (~8-9pt)
- Separated by a thin horizontal rule below the text
- No header on cover page

### Footer
- Bottom-left: Date (e.g., "18 December 2025")
- Bottom-right: Page number
- Separator: Thin horizontal rule above footer content on body pages

### Cover Page Edge Text
- Left edge (vertical): "For the exclusive use of [CLIENT]" — small text, rotated 90 degrees, gray

## 6. Cover Page Design (2026+ Style)

### CRITICAL: NO Dark Gradient Bars

The 2026 edition uses a completely clean white background cover. Earlier 2024/2025 editions used dark navy gradient bars — this is NOT the target style.

### Layout (Top to Bottom)

1. **Brand header** (top-left):
   - Firm name in stacked wordmark style (two-line layout)
   - "Research" or "Commodities Research" beside the stacked wordmark
   - Clean text only on white background

2. **Date/time** (top-right):
   - "Commodities Research" in bold/strong text
   - Below it: "18 December 2025 | 11:33AM EST"

3. **Vertical accent line** (left margin):
   - Thin vertical navy bar/line below the brand header
   - Extends partway down the left side

4. **Series label**: "COMMODITY VIEWS" (bold, smaller, all-caps or small-caps)

5. **Report title**: Large bold title (e.g., "2026 Outlook: Ride the Power Race and Supply Waves")

6. **Horizontal separator**: Thin gray line separating header from content area

7. **Two-column layout below separator**:
   - **Left column** (~60% width): 5-8 key takeaway bullets with filled-square markers
     - Main bullets: large filled squares
     - Sub-bullets: empty squares with bold trade names (e.g., "**Our long gold trade** recommendation...")
   - **Right column** (~40% width): Author team listing
     - Each analyst: **Name** | Phone | Email | Entity

8. **Top Trades section** (below two-column area, spans full width):
   - "Top trades:" with filled square
   - 3-4 trades with empty-square checkboxes, bold trade name, brief description

9. **Disclaimer bar** (bottom):
   - Thin line separator
   - Small regulatory text: "Investors should consider this report as only a single factor..."
   - Link to disclosure appendix

## 7. Body Page Design

### First Body Page (page 2)
- Running header present
- Report title repeated at top: "2026 Outlook: Ride the Power Race and Supply Waves"
- Opening thesis paragraph (4-6 sentences)
- Bullet-point overview of key themes
- First exhibits appear inline (Exhibit 1, Exhibit 2, etc.)

### Typical Body Page Flow
- Running header at top
- Text paragraphs with inline bold subheadings marking new themes
- Exhibits appear between paragraphs (not in separate sections)
- Footnotes at bottom of page with superscript numbering
- Date + page number at footer

### Subheading Style in Text
- Bold text at start of paragraph or standalone line
- Examples from reference: "**Long Gold**", "**Insurance Value of Commodities**", "**Long Copper, Short Aluminum**", italic descriptive line below (e.g., "*Copper Price Consolidates in 2026 as Tariffs Boost Appears Priced in*")
- Followed immediately by analytical paragraphs

### Cross-Reference Style
- Exhibits: underlined and/or blue "(Exhibit N)" or "Exhibit N, left/right panel"
- External links: underlined text with hyperlink styling
- Prior forecast comparisons: "vs. $98 prior" inline
