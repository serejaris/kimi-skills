## PSNR（峰值信噪比）

### 定义
PSNR 衡量信号最大功率与噪声功率之间的比值。它常用于衡量有损压缩编解码器的重建质量。

### 量表
- **范围**：通常 20-50 dB
- **越高越好**：信号越多、噪声越少

### 画质等级对照

| PSNR (dB) | 画质等级 | 适用场景 |
|-----------|----------|----------|
| < 20 | 差 | 大多数场景不可接受 |
| 20-25 | 低 | 仅适用于极低带宽场景 |
| 25-30 | 一般 | 基本视频流 |
| 30-35 | 良好 | 标准流媒体画质 |
| 35-40 | 很好 | 高画质流媒体 |
| 40+ | 优秀 | 接近无损，适合存档 |

### 计算公式
```
PSNR = 10 * log10(MAX_I^2 / MSE)
```
其中：
- MAX_I = 最大像素值（8 位图像为 255）
- MSE = 均方误差

## SSIM（结构相似性指数）

### 定义
SSIM 是一种感知指标，基于结构信息变化而非像素级差异来量化图像质量退化程度。

### 量表
- **范围**：0.0 到 1.0
- **越高越好**：结构相似度越高

### 画质等级对照

| SSIM | 画质等级 | 适用场景 |
|------|----------|----------|
| < 0.70 | 差 | 可见伪影，结构损伤 |
| 0.70-0.80 | 一般 | 明显可感知的画质损失 |
| 0.80-0.90 | 良好 | 大多数流媒体可接受 |
| 0.90-0.95 | 很好 | 高画质流媒体 |
| 0.95-0.98 | 优秀 | 感知上几乎相同 |
| 0.98+ | 完美 | 与原始画面无法区分 |

### 组成部分
SSIM 综合了三项对比：
1. **亮度**：局部亮度对比
2. **对比度**：局部对比度对比
3. **结构**：局部结构相关性

## VMAF（视频多方法评估融合）

### 定义
VMAF 是一种基于机器学习的指标，通过融合多种画质指标来预测主观视频画质。

### 量表
- **范围**：0-100
- **越高越好**：感知画质越好

### 画质等级对照

| VMAF | 画质等级 | 适用场景 |
|------|----------|----------|
| < 20 | 差 | 不可接受 |
| 20-40 | 低 | 基本流媒体 |
| 40-60 | 一般 | 标准流媒体 |
| 60-80 | 良好 | 高画质流媒体 |
| 80-90 | 很好 | 高端流媒体 |
| 90+ | 优秀 | 参考级画质 |

## 文件大小和码率参考

### 按场景划分的压缩目标

| 使用场景 | 文件缩减 | PSNR 目标 | SSIM 目标 |
|----------|----------|-----------|-----------|
| 社交媒体 | 40-60% | 35-40 dB | 0.95-0.98 |
| 流媒体 | 50-70% | 30-35 dB | 0.90-0.95 |
| 存档 | 20-40% | 40+ dB | 0.98+ |
| 移动端 | 60-80% | 25-30 dB | 0.85-0.90 |

### 码率参考

| 分辨率 | 目标码率（以 1080p 为基准） |
|--------|---------------------------|
| 480p | 1-2 Mbps |
| 720p | 2-5 Mbps |
| 1080p | 5-10 Mbps |
| 4K | 20-50 Mbps |
